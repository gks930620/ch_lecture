# 2.3 형변환 (Type Casting)

## 학습 목표
- 자동 형변환과 강제 형변환의 차이를 이해한다
- 형변환 시 주의사항을 안다

---

## 1. 형변환이란?

- 하나의 타입을 다른 타입으로 변환
- **자동 형변환**: 작은 타입 → 큰 타입 (안전)
- **강제 형변환**: 큰 타입 → 작은 타입 (데이터 손실 가능)

---

## 2. 자동 형변환 (Implicit Casting)

### 2.1 자동 형변환 방향
```
byte → short → int → long → float → double
         char → int
```

```java
byte b = 10;
int i = b;      // byte → int (자동)
long l = i;     // int → long (자동)
float f = l;    // long → float (자동)
double d = f;   // float → double (자동)

System.out.println("byte: " + b);
System.out.println("int: " + i);
System.out.println("long: " + l);
System.out.println("float: " + f);
System.out.println("double: " + d);

```

### 2.2 연산 시 자동 형변환
```java
int i = 10;
double d = 3.14;

double result = i + d;  // int가 double로 자동 변환
System.out.println(result);  // 13.14

int a = 10;
int b = 3;
double div = (double) a / b;  // 3.333... (명시적 변환)
System.out.println(div);
```
---

## 3. 강제 형변환 (Explicit Casting)

### 3.1 큰 타입 → 작은 타입
```java
double d = 3.14;
int i = (int) d;  // 소수점 버림
System.out.println(i);  // 3

long l = 1000L;
int i2 = (int) l;
System.out.println(i2);  // 1000

// 주의: 범위 초과 시 오버플로우
int big = 1000;
byte b = (byte) big;  // -24 (오버플로우 발생!)
System.out.println(b);
```

### 3.2 char와 int 변환
```java
char c = 'A';
int i = c;  // 65 (ASCII 코드)
System.out.println(i);

int num = 66;
char ch = (char) num;  // 'B'
System.out.println(ch);
```

---

## 4. 주의사항

### 4.1 데이터 손실
```java
double d = 9.99;
int i = (int) d;  // 9 (소수점 버림)

int big = 300;
byte b = (byte) big;  // 44 (오버플로우)
```

### 4.2 연산 시 주의
```java
int a = 10;
int b = 3;
double result1 = a / b;  // 3.0 (정수 나눗셈 후 double 변환)
double result2 = (double) a / b;  // 3.333... (올바른 방법)
```

---

## 5. 정리

### 핵심 개념
✅ **자동 형변환**: 작은 타입 → 큰 타입 (안전)  
✅ **강제 형변환**: 큰 타입 → 작은 타입 (명시적, 손실 가능)  
✅ **오버플로우**: 범위 초과 시 예상치 못한 값  
✅ **연산 시 주의**: 정수 나눗셈은 정수 결과
