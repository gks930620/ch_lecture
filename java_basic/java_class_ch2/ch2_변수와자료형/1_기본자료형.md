# 2.1 기본 자료형 (Primitive Types)

## 학습 목표
- Java의 8가지 기본 자료형을 이해한다
- 각 자료형의 크기와 범위를 안다
- 적절한 자료형을 선택하여 사용할 수 있다

---

## 1. 기본 자료형이란?

### 1.1 기본 자료형 (Primitive Type)
- Java에서 **미리 정의되어 있는 자료형**
- 총 **8가지**
- 실제 값을 메모리에 저장
- null을 가질 수 없음

---

## 2. 8가지 기본 자료형

### 2.1 정수형 (4가지)

| 자료형 | 크기 | 범위 | 기본값 |
|--------|------|------|--------|
| **byte** | 1 byte (8 bit) | -128 ~ 127 | 0 |
| **short** | 2 byte (16 bit) | -32,768 ~ 32,767 | 0 |
| **int** | 4 byte (32 bit) | -2,147,483,648 ~ 2,147,483,647 | 0 |
| **long** | 8 byte (64 bit) | -9,223,372,036,854,775,808 ~ 9,223,372,036,854,775,807 | 0L |

#### 2.1.1 byte의 8비트 구성과 음수 표현
- `byte`는 8개의 비트(0 또는 1)로 값을 표현한다.
- Java의 정수형(`byte`, `short`, `int`, `long`)은 모두 **2의 보수(Two's Complement)** 방식으로 음수를 저장한다.
- 8비트에서 맨 앞 비트(MSB)는 부호 해석에 사용되며, `0`이면 0 또는 양수, `1`이면 음수로 해석된다.
- 양수는 일반 이진수와 동일하게 저장되고, 음수는 "비트 반전 후 1을 더한 값"으로 표현된다. (2의 보수)
  (핵심 : -1과 1을 더하면 0이 되어야한다)


```java
byte b = 127;           // 작은 범위의 정수
short s = 32000;        // 중간 범위의 정수
int i = 1000000;        // 가장 많이 사용 (기본)
long l = 10000000000L;  // 큰 범위의 정수 (L 붙임)
```

⚠️ **주의:** long 타입은 숫자 뒤에 **L** 또는 **l**을 붙여야 함 (권장: 대문자 L)

### 2.2 실수형 (2가지)

| 자료형 | 크기 | 정밀도 | 기본값 |
|--------|------|--------|--------|
| **float** | 4 byte (32 bit) | 소수점 7자리 | 0.0f |
| **double** | 8 byte (64 bit) | 소수점 15자리 | 0.0d |

#### 2.2.1 실수의 부동소수점(IEEE 754) 표현
- `float`(32비트), `double`(64비트)는 실수를 **부동소수점(Floating Point)** 방식으로 저장한다.
- 값은 크게 **부호(Sign) + 지수(Exponent) + 가수(Fraction/Mantissa)** 세 부분으로 나뉘어 저장된다.
- 지수를 사용해 소수점 위치를 움직이므로, 매우 큰 수와 매우 작은 수를 모두 표현할 수 있다.
- 다만 많은 10진 소수(예: 0.1)는 2진수로 정확히 떨어지지 않아 **근사값**으로 저장된다.
- 따라서 실수 계산에서는 반올림 오차가 생길 수 있고, 값 비교 시 오차 허용 범위를 고려해야 한다.

```java
float f = 3.14f;        // f 또는 F 붙임
double d = 3.141592;    // 실수의 기본형 (d 생략 가능)
```

⚠️ **주의:** float 타입은 숫자 뒤에 **f** 또는 **F**를 붙여야 함

### 2.3 문자형 (1가지)

| 자료형 | 크기 | 범위 | 기본값 |
|--------|------|------|--------|
| **char** | 2 byte (16 bit) | 0 ~ 65,535 (유니코드) | '\u0000' |

```java
char c1 = 'A';          // 문자 (작은 따옴표)
char c2 = '가';         // 한글도 가능
char c3 = 65;           // 숫자로도 표현 가능 (ASCII 코드)
char c4 = '\u0041';     // 유니코드로 표현 (A)
```

### 2.4 논리형 (1가지)

| 자료형 | 크기 | 값 | 기본값 |
|--------|------|-----|--------|
| **boolean** | 1 bit (이론상) | true, false | false |

```java
boolean isTrue = true;
boolean isFalse = false;
```

## 3. 자료형 선택 가이드

### 3.1 정수를 사용할 때
```java
int age = 25;              // 일반적인 정수 → int
long population = 7800000000L;  // 매우 큰 수 → long
byte score = 100;          // -128~127 범위 → byte
```

💡 **권장:** 특별한 이유가 없다면 **int** 사용

### 3.2 실수를 사용할 때
```java
double pi = 3.141592;      // 일반적인 실수 → double
float height = 175.5f;     // 메모리 절약이 필요할 때 → float
```

💡 **권장:** 일반적으로 **double** 사용 (더 정밀함)

## 4. 실습 예제

### 예제 1: 기본 자료형 사용
```java
package ch2_변수와자료형.part1_기본자료형;

public class PrimitiveTypes {
    public static void main(String[] args) {
        // 정수형
        byte b = 127;
        short s = 32000;
        int i = 2147483647;
        long l = 9223372036854775807L;
        
        System.out.println("=== 정수형 ===");
        System.out.println("byte: " + b);
        System.out.println("short: " + s);
        System.out.println("int: " + i);
        System.out.println("long: " + l);
        
        // 실수형
        float f = 3.14f;
        double d = 3.141592653589793;
        
        System.out.println("\n=== 실수형 ===");
        System.out.println("float: " + f);
        System.out.println("double: " + d);
        
        // 문자형
        char c1 = 'A';
        char c2 = '가';
        char c3 = 65;
        
        System.out.println("\n=== 문자형 ===");
        System.out.println("char c1: " + c1);
        System.out.println("char c2: " + c2);
        System.out.println("char c3 (65): " + c3);
        
        // 논리형
        boolean isJavaFun = true;
        
        System.out.println("\n=== 논리형 ===");
        System.out.println("boolean: " + isJavaFun);
    }
}
```

### 예제 2: 자료형 크기 확인
```java
package ch2_변수와자료형.part1_기본자료형;

public class DataTypeSize {
    public static void main(String[] args) {
        System.out.println("=== 기본 자료형 크기 ===");
        System.out.println("byte: " + Byte.BYTES + " bytes");
        System.out.println("short: " + Short.BYTES + " bytes");
        System.out.println("int: " + Integer.BYTES + " bytes");
        System.out.println("long: " + Long.BYTES + " bytes");
        System.out.println("float: " + Float.BYTES + " bytes");
        System.out.println("double: " + Double.BYTES + " bytes");
        System.out.println("char: " + Character.BYTES + " bytes");
        
        System.out.println("\n=== 정수형 범위 ===");
        System.out.println("byte: " + Byte.MIN_VALUE + " ~ " + Byte.MAX_VALUE);
        System.out.println("short: " + Short.MIN_VALUE + " ~ " + Short.MAX_VALUE);
        System.out.println("int: " + Integer.MIN_VALUE + " ~ " + Integer.MAX_VALUE);
        System.out.println("long: " + Long.MIN_VALUE + " ~ " + Long.MAX_VALUE);
    }
}
```

## 5. 주의사항

### 5.1 오버플로우 (Overflow)
- 정수형은 비트 수가 고정되어 있어, 표현 가능한 최대값을 넘으면 상위 비트가 버려지고 하위 비트만 남는다.
- 이 동작은 비트 패턴이 순환하는 형태이며, `byte`에서 `127` 다음 값이 `-128`이 되는 이유다.
- 반대로 최소값보다 더 작아지는 경우도 같은 원리로 최대값 쪽으로 순환한다(언더플로우).
- Java에서 정수 오버플로우/언더플로우는 예외를 던지지 않는 **정상 동작**이다.
- 이런 순환 해석이 가능한 이유가 바로 정수의 음수 표현 방식인 **2의 보수**다.

```java
byte b = 127;
b++;  // -128이 됨 (오버플로우)
```

### 5.2 언더플로우 (Underflow)
```java
byte b = -128;
b--;  // 127이 됨 (언더플로우)
```

## 6. 정리

### 핵심 개념
✅ Java는 8가지 기본 자료형을 제공  
✅ 정수는 **int**, 실수는 **double**이 기본  
✅ long은 **L**, float는 **f** 접미사 필수  
✅ char는 2 byte (유니코드 지원)  
✅ boolean은 true/false만 가능  

### 자주 사용하는 자료형
- **int**: 정수 (가장 많이 사용)
- **double**: 실수
- **boolean**: 참/거짓
- **char**: 문자 1개
