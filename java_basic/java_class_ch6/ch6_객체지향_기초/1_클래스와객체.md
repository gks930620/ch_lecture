## 1. 클래스와 객체

### 1.1 클래스란?
- **객체를 만들기 위한 설계도, 틀**
- 속성(필드)과 동작(메소드)을 정의

### 1.2 객체란?
- **클래스로부터 생성된 실체**
- 메모리에 할당된 인스턴스

```java
// 클래스 정의
public class Person {
    // 필드 (속성)
    String name;
    int age;
    
    // 메소드 (동작)
    void introduce() {
        System.out.println("이름: " + name + ", 나이: " + age);
    }
}

// 객체 생성
Person person1 = new Person();
person1.name = "홍길동";
person1.age = 25;
person1.introduce();
```

## 2. 생성자 (Constructor)

### 2.1 생성자란?
- 객체가 생성될 때 **자동으로 호출**되는 특별한 메소드
- 클래스 이름과 동일
- 반환 타입이 없음

```java
public class Person {
    String name;
    int age;
    
    // 기본 생성자
    public Person() {
        System.out.println("Person 객체 생성");
    }
    
    // 매개변수가 있는 생성자
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

// 사용
Person p1 = new Person();                    // 기본 생성자
Person p2 = new Person("홍길동", 25);        // 매개변수 생성자
```

## 3. this 키워드

```java
public class Person {
    String name;
    
    public Person(String name) {
        this.name = name;  // this.name은 필드, name은 매개변수
    }
    
    public void setName(String name) {
        this.name = name;
    }
}
```

## 4. 접근 제어자

| 접근 제어자 | 같은 클래스 | 같은 패키지 | 자식 클래스 | 전체 |
|------------|-----------|-----------|-----------|------|
| **public** | ✅ | ✅ | ✅ | ✅ |
| **protected** | ✅ | ✅ | ✅ | ❌ |
| **default** | ✅ | ✅ | ❌ | ❌ |
| **private** | ✅ | ❌ | ❌ | ❌ |

```java
public class Person {
    public String name;       // 어디서나 접근 가능
    protected int age;        // 같은 패키지 + 자식 클래스
    String address;           // 같은 패키지만 (default)
    private String ssn;       // 클래스 내부만
    
    // Getter/Setter
    public String getSsn() {
        return ssn;
    }
    
    public void setSsn(String ssn) {
        this.ssn = ssn;
    }
}
```

## 5. Getter / Setter

```java
public class Person {
    private String name;
    private int age;
    
    // Getter
    public String getName() {
        return name;
    }
    
    // Setter
    public void setName(String name) {
        this.name = name;
    }
    
    public int getAge() {
        return age;
    }
    
    public void setAge(int age) {
        if (age > 0) {  // 유효성 검사
            this.age = age;
        }
    }
}
```

## 6. static 키워드

### 6.1 정적 변수 (클래스 변수)
```java
public class Counter {
    static int count = 0;  // 모든 객체가 공유
    
    public Counter() {
        count++;
    }
}

Counter c1 = new Counter();
Counter c2 = new Counter();
System.out.println(Counter.count);  // 2
```

### 6.2 정적 메소드
```java
public class MathUtil {
    public static int add(int a, int b) {
        return a + b;
    }
}

// 객체 생성 없이 호출
int result = MathUtil.add(10, 20);
```

## 7. 싱글톤 패턴

```java
public class Singleton {
    private static Singleton instance;
    
    // private 생성자
    private Singleton() {
    }
    
    // 유일한 인스턴스 반환
    public static Singleton getInstance() {
        if (instance == null) {
            instance = new Singleton();
        }
        return instance;
    }
}

// 사용
Singleton s1 = Singleton.getInstance();
Singleton s2 = Singleton.getInstance();
System.out.println(s1 == s2);  // true (같은 객체)
```

## 8. 정리

### 핵심 개념
✅ **클래스**: 객체의 설계도  
✅ **객체**: 클래스로부터 생성된 실체  
✅ **생성자**: 객체 초기화  
✅ **this**: 현재 객체 참조  
✅ **접근 제어자**: public, private, protected, default  
✅ **static**: 클래스 레벨에서 공유
