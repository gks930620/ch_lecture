# 10장. 문자열 처리

## 학습 목표
- String 클래스의 다양한 메소드를 활용할 수 있다
- String의 불변성을 이해한다
- StringBuilder와 StringBuffer의 차이를 안다

---

## 1. String 불변성 (Immutability)

```java
String str = "Hello";
str = str + " World";  // 새로운 String 객체 생성

// String은 불변이므로 원본은 변경되지 않음
String s1 = "Java";
String s2 = s1.toUpperCase();  // 새 객체 반환
System.out.println(s1);  // "Java" (원본 유지)
System.out.println(s2);  // "JAVA"
```

## 2. String 주요 메소드

```java
String str = "Hello World";

// 길이
str.length();  // 11

// 문자 추출
str.charAt(0);  // 'H'

// 부분 문자열
str.substring(0, 5);  // "Hello"
str.substring(6);     // "World"

// 대소문자 변환
str.toLowerCase();  // "hello world"
str.toUpperCase();  // "HELLO WORLD"

// 포함 여부
str.contains("World");    // true
str.startsWith("Hello");  // true
str.endsWith("World");    // true

// 문자열 교체
str.replace("World", "Java");  // "Hello Java"
str.replaceAll("\\s", "");     // 공백 제거 (정규식)

// 공백 제거
"  Hello  ".trim();  // "Hello"
"  Hello  ".strip(); // "Hello" (Java 11+, 유니코드 지원)

// 분리
str.split(" ");  // ["Hello", "World"]

// 비교
str.equals("Hello World");  // true
str.equalsIgnoreCase("hello world");  // true
str.compareTo("ABC");  // 양수 (사전순)

// 찾기
str.indexOf("o");      // 4 (첫 번째 위치)
str.lastIndexOf("o");  // 7 (마지막 위치)
```

## 3. StringBuilder vs StringBuffer

### 3.1 StringBuilder (권장)
```java
StringBuilder sb = new StringBuilder();
sb.append("Hello");
sb.append(" ");
sb.append("World");
sb.insert(5, ",");  // "Hello, World"
sb.delete(5, 6);    // "Hello World"
sb.reverse();       // "dlroW olleH"
String result = sb.toString();
```

### 3.2 StringBuffer (스레드 안전)
```java
StringBuffer sbf = new StringBuffer();
sbf.append("Thread");
sbf.append(" Safe");
```

| 구분 | String | StringBuilder | StringBuffer |
|------|--------|--------------|-------------|
| **불변성** | 불변 | 가변 | 가변 |
| **스레드 안전** | O | X | O |
| **성능** | 느림 | 빠름 | 보통 |
| **사용 시기** | 변경 없을 때 | 단일 스레드 | 멀티 스레드 |

## 4. String Pool

```java
String s1 = "Hello";      // String Pool
String s2 = "Hello";      // 같은 객체 참조
String s3 = new String("Hello");  // Heap 영역

System.out.println(s1 == s2);  // true
System.out.println(s1 == s3);  // false
System.out.println(s1.equals(s3));  // true
```

## 5. 문자열 포맷팅

```java
// String.format
String formatted = String.format("이름: %s, 나이: %d", "홍길동", 25);

// printf
System.out.printf("%.2f%%\n", 85.5);  // "85.50%"

// 포맷 지정자
%s  // 문자열
%d  // 정수
%f  // 실수
%c  // 문자
%b  // boolean
```

## 7. 정리

### 핵심 개념
✅ **String은 불변** (변경 시 새 객체 생성)  
✅ **StringBuilder**: 가변 문자열 (단일 스레드)  
✅ **StringBuffer**: 가변 문자열 (멀티 스레드)  
✅ **String Pool**: 리터럴 문자열 재사용  
✅ **equals()**: 내용 비교, ==: 참조 비교
