## 2. 조건문

### 2.1 if문
```java
int score = 85;

if (score >= 90) {
    System.out.println("A학점");
} else if (score >= 80) {
    System.out.println("B학점");
} else if (score >= 70) {
    System.out.println("C학점");
} else {
    System.out.println("F학점");
}
```

### 2.2 switch문 (기본)
```java
int day = 3;

switch (day) {
    case 1:
        System.out.println("월요일");
        break;
    case 2:
        System.out.println("화요일");
        break;
    case 3:
        System.out.println("수요일");
        break;
    default:
        System.out.println("기타");
}
```

### 2.3 switch 표현식 (Java 14+)
```java
int day = 3;
String dayName = switch (day) {
    case 1 -> "월요일";
    case 2 -> "화요일";
    case 3 -> "수요일";
    case 4 -> "목요일";
    case 5 -> "금요일";
    default -> "주말";
};
System.out.println(dayName);  // "수요일"
```

## 3. 반복문

### 3.1 for문
```java
// 기본 for문
for (int i = 1; i <= 5; i++) {
    System.out.println(i);
}

// 배열 순회
int[] numbers = {10, 20, 30, 40, 50};
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// 향상된 for문 (for-each)
for (int num : numbers) {
    System.out.println(num);
}
```

### 3.2 while문
```java
int i = 1;
while (i <= 5) {
    System.out.println(i);
    i++;
}

// 무한 루프
while (true) {
    // break로 탈출
}
```

### 3.3 do-while문
```java
int i = 1;
do {
    System.out.println(i);
    i++;
} while (i <= 5);

// 최소 1번은 실행됨
```

### 3.4 break와 continue
```java
// break: 반복문 탈출
for (int i = 1; i <= 10; i++) {
    if (i == 5) break;
    System.out.println(i);  // 1, 2, 3, 4
}

// continue: 다음 반복으로
for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);  // 1, 2, 4, 5
}
```

### 3.5 레이블 (label)
```java
outer:
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        if (i == 2 && j == 2) {
            break outer;  // 외부 반복문 탈출
        }
        System.out.println(i + ", " + j);
    }
}
```

## 4. 정리

### 연산자 우선순위 (높은 순)
1. `()` - 괄호
2. `++`, `--`, `!` - 단항 연산자
3. `*`, `/`, `%` - 곱셈, 나눗셈
4. `+`, `-` - 덧셈, 뺄셈
5. `<`, `>`, `<=`, `>=` - 비교 연산자
6. `==`, `!=` - 같음, 다름
7. `&&` - AND
8. `||` - OR
9. `? :` - 삼항 연산자
10. `=`, `+=`, `-=` - 대입 연산자

### 핵심 개념
✅ **산술 연산자**: +, -, *, /, %  
✅ **비교 연산자**: ==, !=, >, <, >=, <=  
✅ **논리 연산자**: &&, ||, !  
✅ **조건문**: if-else, switch  
✅ **반복문**: for, while, do-while  
✅ **제어**: break, continue
