# 16장. 멀티스레드 (Multithreading)

- Thread 클래스와 Runnable 인터페이스를 사용할 수 있다
- 동기화를 이해한다

---

## 1. 스레드 생성

### 1.1 Thread 클래스 상속
```java
class MyThread extends Thread {
    @Override
    public void run() {
        for (int i = 0; i < 5; i++) {
            System.out.println(getName() + ": " + i);
        }
    }
}

MyThread t = new MyThread();
t.start();  // run() 메소드 실행
```

### 1.2 Runnable 인터페이스 구현
```java
class MyRunnable implements Runnable {
    @Override
    public void run() {
        for (int i = 0; i < 5; i++) {
            System.out.println(Thread.currentThread().getName() + ": " + i);
        }
    }
}

Thread t = new Thread(new MyRunnable());
t.start();
```

### 1.3 람다식
```java
Thread t = new Thread(() -> {
    for (int i = 0; i < 5; i++) {
        System.out.println(i);
    }
});
t.start();
```

## 2. 동기화 (synchronized)

```java
class Counter {
    private int count = 0;
    
    public synchronized void increment() {
        count++;
    }
    
    public int getCount() {
        return count;
    }
}
```

---

## 3. 정리

### 핵심 개념
✅ **Thread**: 프로그램 실행 흐름  
✅ **Runnable**: 스레드 작업 정의  
✅ **synchronized**: 동기화 (thread-safe)  
✅ **start()**: 스레드 시작  
✅ **join()**: 스레드 종료 대기
