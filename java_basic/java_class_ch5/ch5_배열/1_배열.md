## 1. 배열의 기본

### 1.1 배열이란?
- **같은 타입**의 데이터를 여러 개 저장하는 자료구조
- 고정된 크기
- 인덱스는 0부터 시작

### 1.2 1차원 배열
```java
// 선언과 생성
int[] arr1 = new int[5];
int[] arr2 = {1, 2, 3, 4, 5};
int[] arr3 = new int[] {1, 2, 3, 4, 5};

// 접근
arr1[0] = 10;
int value = arr1[0];

// 길이
int length = arr1.length;
```

## 2. 다차원 배열

### 2.1 2차원 배열
```java
// 선언
int[][] arr = new int[3][4];  // 3행 4열

// 초기화
int[][] arr2 = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// 접근
arr[0][0] = 10;
int value = arr[1][2];  // 6

// 순회
for (int i = 0; i < arr2.length; i++) {
    for (int j = 0; j < arr2[i].length; j++) {
        System.out.print(arr2[i][j] + " ");
    }
    System.out.println();
}
```

## 3. 배열 복사

### 3.1 얕은 복사 (참조 복사)
```java
int[] arr1 = {1, 2, 3};
int[] arr2 = arr1;  // 같은 배열을 가리킴
arr2[0] = 100;
System.out.println(arr1[0]);  // 100
```

### 3.2 깊은 복사 (값 복사)
```java
// 반복문
int[] arr1 = {1, 2, 3};
int[] arr2 = new int[arr1.length];
for (int i = 0; i < arr1.length; i++) {
    arr2[i] = arr1[i];
}

// System.arraycopy
System.arraycopy(arr1, 0, arr2, 0, arr1.length);

// Arrays.copyOf
int[] arr3 = Arrays.copyOf(arr1, arr1.length);

// clone
int[] arr4 = arr1.clone();
```

## 4. Arrays 클래스

```java
import java.util.Arrays;

int[] arr = {5, 2, 8, 1, 9};

// 정렬
Arrays.sort(arr);  // {1, 2, 5, 8, 9}

// 이진 탐색 (정렬 필수)
int index = Arrays.binarySearch(arr, 5);  // 2

// 배열을 문자열로
String str = Arrays.toString(arr);  // "[1, 2, 5, 8, 9]"

// 배열 채우기
Arrays.fill(arr, 0);  // 모든 요소를 0으로

// 배열 비교
boolean equal = Arrays.equals(arr1, arr2);
```

## 5. 향상된 for문 (for-each)

```java
int[] numbers = {1, 2, 3, 4, 5};

// 일반 for문
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// 향상된 for문 (읽기 전용)
for (int num : numbers) {
    System.out.println(num);
}
```

## 6. 정리

### 핵심 개념
✅ 배열은 **고정 크기**의 자료구조  
✅ 인덱스는 **0부터 시작**  
✅ `length` 프로퍼티로 길이 확인  
✅ 깊은 복사가 필요한 경우 `clone()` 또는 `Arrays.copyOf()` 사용  
✅ `Arrays` 클래스로 정렬, 검색, 변환 등 수행
