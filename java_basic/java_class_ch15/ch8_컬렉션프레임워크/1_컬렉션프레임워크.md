## 1. 컬렉션 프레임워크란?

- 데이터를 저장하는 자료구조와 알고리즘을 구현한 클래스 집합
- `java.util` 패키지

```
Collection
├── List (순서O, 중복O)
│   ├── ArrayList
│   ├── LinkedList
│   └── Vector
├── Set (순서X, 중복X)
│   ├── HashSet
│   ├── TreeSet
│   └── LinkedHashSet
└── Queue
    ├── PriorityQueue
    └── Deque (ArrayDeque)

Map (Key-Value, Key 중복X)
├── HashMap
├── TreeMap
└── LinkedHashMap
```

## 2. List 인터페이스

### 2.1 ArrayList
```java
ArrayList<String> list = new ArrayList<>();

// 추가
list.add("Apple");
list.add("Banana");
list.add(1, "Cherry");  // 인덱스 지정

// 조회
String item = list.get(0);
int size = list.size();

// 삭제
list.remove(0);
list.remove("Banana");

// 순회
for (String fruit : list) {
    System.out.println(fruit);
}
```

### 2.2 LinkedList
- 삽입/삭제가 빈번할 때 유리
```java
LinkedList<String> linkedList = new LinkedList<>();
linkedList.add("First");
linkedList.addFirst("New First");
linkedList.addLast("Last");
```

## 3. Set 인터페이스

### 3.1 HashSet
```java
HashSet<String> set = new HashSet<>();

set.add("Apple");
set.add("Banana");
set.add("Apple");  // 중복 무시됨

System.out.println(set.size());  // 2
System.out.println(set.contains("Apple"));  // true

for (String item : set) {
    System.out.println(item);  // 순서 보장 안 됨
}
```

### 3.2 TreeSet (정렬됨)
```java
TreeSet<Integer> treeSet = new TreeSet<>();
treeSet.add(5);
treeSet.add(2);
treeSet.add(8);
// 출력: 2, 5, 8 (자동 정렬)
```

## 4. Map 인터페이스

### 4.1 HashMap
```java
HashMap<String, Integer> map = new HashMap<>();

// 추가
map.put("Apple", 1000);
map.put("Banana", 2000);
map.put("Cherry", 3000);

// 조회
int price = map.get("Apple");  // 1000

// 존재 여부
boolean exists = map.containsKey("Apple");

// 삭제
map.remove("Banana");

// 순회
for (String key : map.keySet()) {
    System.out.println(key + ": " + map.get(key));
}

for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}
```

## 5. Iterator

ArrayList<String> list = new ArrayList<>();

## 6. Comparable vs Comparator

### 6.1 Comparable
```java
class Student implements Comparable<Student> {
    String name;
    int score;
    
    @Override
    public int compareTo(Student other) {
        return this.score - other.score;  // 점수 오름차순
    }
}

Collections.sort(studentList);
```

### 6.2 Comparator
```java
Collections.sort(studentList, new Comparator<Student>() {
    @Override
    public int compare(Student s1, Student s2) {
        return s2.score - s1.score;  // 점수 내림차순
    }
});

// 람다식
Collections.sort(studentList, (s1, s2) -> s2.score - s1.score);
```

## 7. Collections 유틸리티

```java
// 정렬
Collections.sort(list);
Collections.reverse(list);

// 섞기
Collections.shuffle(list);

// 최댓값, 최솟값
int max = Collections.max(list);
int min = Collections.min(list);

// 빈도수
int count = Collections.frequency(list, "Apple");
```

## 8. 정리

### 핵심 개념
✅ **List**: 순서O, 중복O (ArrayList, LinkedList)  
✅ **Set**: 순서X, 중복X (HashSet, TreeSet)  
✅ **Map**: Key-Value (HashMap, TreeMap)  
✅ **Iterator**: 컬렉션 순회  
✅ **Comparable/Comparator**: 정렬 기준 정의
