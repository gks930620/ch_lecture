## 1. 상속 (Inheritance)

### 1.1 상속이란?
- 기존 클래스의 필드와 메소드를 **재사용**
- `extends` 키워드 사용
- Java는 **단일 상속**만 지원

```java
// 부모 클래스
public class Animal {
    protected String name;
    
    public void eat() {
        System.out.println(name + "이(가) 먹습니다.");
    }
}

// 자식 클래스
public class Dog extends Animal {
    public void bark() {
        System.out.println(name + "이(가) 짖습니다.");
    }
}

Dog dog = new Dog();
dog.name = "멍멍이";
dog.eat();   // 부모 메소드
dog.bark();  // 자식 메소드
```

## 2. 메소드 오버라이딩

```java
public class Animal {
    public void sound() {
        System.out.println("동물 소리");
    }
}

public class Dog extends Animal {
    @Override  // 오버라이딩 명시
    public void sound() {
        System.out.println("멍멍!");
    }
}

Animal animal = new Dog();
animal.sound();  // "멍멍!" (오버라이딩된 메소드 호출)
```

## 3. super 키워드

```java
public class Parent {
    protected String name;
    
    public Parent(String name) {
        this.name = name;
    }
    
    public void show() {
        System.out.println("부모: " + name);
    }
}

public class Child extends Parent {
    private int age;
    
    public Child(String name, int age) {
        super(name);  // 부모 생성자 호출
        this.age = age;
    }
    
    @Override
    public void show() {
        super.show();  // 부모 메소드 호출
        System.out.println("자식: " + age + "살");
    }
}
```

## 4. 다형성 (Polymorphism)

### 4.1 다형성이란?
- 하나의 타입으로 **여러 종류의 객체**를 다룸

```java
Animal animal1 = new Dog();
Animal animal2 = new Cat();
Animal animal3 = new Bird();

animal1.sound();  // "멍멍!"
animal2.sound();  // "야옹!"
animal3.sound();  // "짹짹!"
```

### 4.2 업캐스팅 / 다운캐스팅
```java
// 업캐스팅 (자동)
Animal animal = new Dog();

// 다운캐스팅 (명시적)
Dog dog = (Dog) animal;
dog.bark();

// instanceof 연산자
if (animal instanceof Dog) {
    Dog d = (Dog) animal;
    d.bark();
}
```

## 5. 추상 클래스 (Abstract Class)

```java
public abstract class Animal {
    protected String name;
    
    // 일반 메소드
    public void eat() {
        System.out.println(name + "이(가) 먹습니다.");
    }
    
    // 추상 메소드 (구현 없음)
    public abstract void sound();
}

public class Dog extends Animal {
    @Override
    public void sound() {
        System.out.println("멍멍!");
    }
}
```

## 6. 인터페이스 (Interface)

### 6.1 인터페이스란?
- 모든 메소드가 추상 메소드 (Java 8 이전)
- `implements` 키워드 사용
- **다중 구현** 가능

```java
public interface Flyable {
    void fly();  // public abstract 생략 가능
}

public interface Swimmable {
    void swim();
}

public class Duck implements Flyable, Swimmable {
    @Override
    public void fly() {
        System.out.println("오리가 날아갑니다.");
    }
    
    @Override
    public void swim() {
        System.out.println("오리가 수영합니다.");
    }
}
```

### 6.2 default 메소드 (Java 8+)
```java
public interface MyInterface {
    default void defaultMethod() {
        System.out.println("기본 구현");
    }
    
    void abstractMethod();
}
```

## 7. Object 클래스

### 7.1 주요 메소드
```java
public class Person {
    private String name;
    private int age;
    
    @Override
    public String toString() {
        return "Person{name='" + name + "', age=" + age + "}";
    }
    
    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;
        if (obj == null || getClass() != obj.getClass()) return false;
        Person person = (Person) obj;
        return age == person.age && name.equals(person.name);
    }
    
    @Override
    public int hashCode() {
        return Objects.hash(name, age);
    }
}
```

## 9. 정리

### 핵심 개념
✅ **상속**: 코드 재사용 (extends)  
✅ **오버라이딩**: 부모 메소드 재정의  
✅ **다형성**: 하나의 타입으로 여러 객체 다루기  
✅ **추상 클래스**: 일부 구현 + 추상 메소드  
✅ **인터페이스**: 명세만 정의 (다중 구현 가능)  
✅ **Object**: 모든 클래스의 최상위 클래스
