## 1. 람다 표현식이란?

- **익명 함수**를 간결하게 표현
- Java 8부터 지원

```java
// 기존 방식
Runnable runnable = new Runnable() {
    @Override
    public void run() {
        System.out.println("Hello");
    }
};

// 람다 표현식
Runnable runnable = () -> System.out.println("Hello");
```

## 2. 람다 문법

```java
// 매개변수 없음
() -> System.out.println("Hello")

// 매개변수 1개
x -> x * 2
(x) -> x * 2

// 매개변수 2개
(x, y) -> x + y

// 여러 줄
(x, y) -> {
    int result = x + y;
    return result;
}
```

## 3. 함수형 인터페이스

- **추상 메소드가 1개**인 인터페이스
- `@FunctionalInterface` 어노테이션 사용 권장

```java
@FunctionalInterface
interface Calculator {
    int calculate(int a, int b);
}

// 사용
Calculator add = (a, b) -> a + b;
Calculator multiply = (a, b) -> a * b;

System.out.println(add.calculate(10, 20));  // 30
System.out.println(multiply.calculate(10, 20));  // 200
```

## 4. java.util.function 패키지

### 4.1 Consumer<T> (매개변수 O, 반환값 X)
```java
Consumer<String> print = str -> System.out.println(str);
print.accept("Hello");
```

### 4.2 Supplier<T> (매개변수 X, 반환값 O)
```java
Supplier<Integer> random = () -> (int)(Math.random() * 100);
System.out.println(random.get());
```

### 4.3 Function<T, R> (매개변수 O, 반환값 O)
```java
Function<String, Integer> strLength = str -> str.length();
System.out.println(strLength.apply("Hello"));  // 5
```

### 4.4 Predicate<T> (매개변수 O, boolean 반환)
```java
Predicate<Integer> isPositive = num -> num > 0;
System.out.println(isPositive.test(10));  // true
System.out.println(isPositive.test(-5));  // false
```

## 5. 메소드 참조

```java
// 람다식
Function<String, Integer> func1 = str -> Integer.parseInt(str);

// 메소드 참조 (정적 메소드)
Function<String, Integer> func2 = Integer::parseInt;

// 인스턴스 메소드
String str = "hello";
Supplier<String> sup1 = () -> str.toUpperCase();
Supplier<String> sup2 = str::toUpperCase;

// 생성자 참조
Supplier<ArrayList<String>> listSup1 = () -> new ArrayList<>();
Supplier<ArrayList<String>> listSup2 = ArrayList::new;
```

## 6. 정리

### 핵심 개념
✅ **람다식**: 익명 함수를 간결하게 표현  
✅ **함수형 인터페이스**: 추상 메소드 1개  
✅ **Consumer**: 소비 (반환값 없음)  
✅ **Supplier**: 공급 (매개변수 없음)  
✅ **Function**: 변환 (입력 → 출력)  
✅ **Predicate**: 검사 (true/false)
