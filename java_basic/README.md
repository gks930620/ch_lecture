## 🎯 학습 방법

### 1️⃣ 각 챕터 구조
- **md 파일**: 이론 설명 및 개념 정리
- **Part1~PartN.java**: 실습 예제 코드 (순서대로 학습)
- **과제_XXX.java**: 챕터별 실전 과제 (도전!)

### 2️⃣ 학습 순서
1. 해당 챕터의 `.md` 파일을 읽고 개념 이해
2. `Part1`, `Part2`, ... 순서대로 예제 코드 실행 및 분석
3. 직접 코드를 수정하며 실험
4. **과제**에 도전하여 실력 점검

### 3️⃣ 과제 안내
각 챕터마다 **1~2개의 실전 과제**가 있습니다.
