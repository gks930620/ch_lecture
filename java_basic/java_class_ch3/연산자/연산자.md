## 1. 연산자

### 1.1 산술 연산자 (+, -, *, /, %)
```java
int a = 10, b = 3;

System.out.println(a + b);  // 13 (덧셈)
System.out.println(a - b);  // 7  (뺄셈)
System.out.println(a * b);  // 30 (곱셈)
System.out.println(a / b);  // 3  (나눗셈 - 정수 나눗셈)
System.out.println(a % b);  // 1  (나머지)

double c = 10.0;
System.out.println(c / b);  // 3.333... (실수 나눗셈)
```

### 1.2 증감 연산자 (++, --)
```java
int x = 10;

// 전위 증감
++x;  // x = 11 (먼저 증가)
--x;  // x = 10 (먼저 감소)

// 후위 증감
x++;  // x = 11 (나중에 증가)
x--;  // x = 10 (나중에 감소)

// 차이점
int y = 10;
int result1 = ++y;  // y=11, result1=11 (먼저 증가)
int z = 10;
int result2 = z++;  // z=11, result2=10 (나중에 증가)
```

### 1.3 비교 연산자 (==, !=, >, <, >=, <=)
```java
int a = 10, b = 20;

System.out.println(a == b);  // false (같다)
System.out.println(a != b);  // true  (다르다)
System.out.println(a > b);   // false (크다)
System.out.println(a < b);   // true  (작다)
System.out.println(a >= b);  // false (크거나 같다)
System.out.println(a <= b);  // true  (작거나 같다)
```

### 1.4 논리 연산자 (&&, ||, !)
```java
boolean t = true, f = false;

System.out.println(t && f);  // false (AND: 둘 다 true일 때만 true)
System.out.println(t || f);  // true  (OR: 하나라도 true면 true)
System.out.println(!t);      // false (NOT: 반대)

// 단축 평가
int a = 10;
if (a > 0 && a < 20) {  // 두 조건 모두 확인
    System.out.println("10 < a < 20");
}
```

### 1.5 대입 연산자 (=, +=, -=, *=, /=, %=)
```java
int x = 10;

x += 5;   // x = x + 5  → 15
x -= 3;   // x = x - 3  → 12
x *= 2;   // x = x * 2  → 24
x /= 4;   // x = x / 4  → 6
x %= 4;   // x = x % 4  → 2
```
