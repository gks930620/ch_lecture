공공데이터 API 프록시 테스트

브라우저(JS)에서 직접 공공데이터 API를 호출하면 CORS 에러가 발생합니다.
우리 Spring 서버가 프록시(대리 호출) 역할을 하여 CORS 문제 없이 데이터를 가져옵니다.
브라우저 JS → 우리 서버(/api/proxy/*) → 공공데이터 API → 응답 → JS
