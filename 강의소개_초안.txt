==================================================
실전 서비스 개발  레슨
==================================================

[🚀 수업 소개]
"문법만 배우는 수업이 아니라,
웹의 동작 원리와  흐름을 이해하고
직접 코딩으로 구현해 실제 서비스 배포까지 연결하는 실전 수업입니다."

취업, 외주, 창업으로 이어질 수 있는
실무형 개발 역량을 중심으로 지도합니다.


[📢 공지 사항]
- 요청 시 연령대, 현재 수준(비전공/전공), 목표(취업/외주/창업)를 함께 알려주세요.
- 외주/서비스 개발 관련 문의도 "프로그래밍/코딩 레슨" 형태로 요청 가능합니다.
- 단건 오류 해결/문제 풀이 요청만으로는 수업 진행이 어렵습니다.
- 과제를 하지 않고서는 다음 수업이 불가능합니다.
- 과제가 어려울 때는 얼마든지  문의하세요.


[🎯 수업 핵심 방향]
- 개발 경험이 거의 없는 입문 상태부터 시작 가능
- 강의 자료 기반 커리큘럼 + 대면 실습
- 개념 설명 후 바로 구현하는 실습 중심 진행
- 웹 기초부터 설계, 구현, 배포까지 하나의 흐름으로 연결
- AI 코딩 도구를 활용한 개발 생산성 향상 노하우 전달
- 현업 실무 개발 및 서비스 구축 경험 기반 1:1 수업
- 비전공자/입문자 대상 실전형 커리큘럼 운영 경험


[🙋 이런 분께 추천합니다]
- 비전공자로 개발을 처음 시작하는 분
- 혼자 공부만으로는 실력이 늘지 않았던 분
- 취업용 포트폴리오를 실전 수준으로 만들고 싶은 분
- 외주/사이드 프로젝트를 직접 진행하고 싶은 분
- 본인 서비스 아이디어를 웹 형태로 구현하고 싶은 분


[🧭 강의 방식]
- 1:1 대면 수업(오프라인)
- 개념 설명 후 코딩으로 바로 구현
- 회차별 코드 리뷰 및 개선 피드백
- 수업 시간 외 질의응답(운영 방식은 별도 협의)
- 필요한 경우 프로젝트 구조, 설계, 배포까지 연결 지도


[📚 수업 커리큘럼]

1) Java 기초부터 실무 문법까지
- Java 기본 문법, 자료형, 연산자, 제어문
- 메소드/스코프, 배열, 객체지향 기초
- 상속, 인터페이스, 예외처리
- Java API(문자열, 날짜/시간, 입출력)
- 제네릭, 컬렉션, 람다, Stream
- JDBC, 최신 문법, 멀티스레드

2) 웹/프론트 기초
- HTML, CSS, JavaScript 기초부터 DOM, 비동기 처리, 웹 요청-응답 통신
- 브라우저 동작 원리 이해
- 실습 중심 프론트엔드 기초 체력 만들기

3) Spring 들어가기 전 (web_basic 핵심 정리)
- WAS/웹서버 개념, 요청-응답 흐름
- JSP 기본 문법과 내장 객체, page directive
- forward/redirect 차이와 동작 원리
- cookie/session 인증 상태 관리 기초
- JSTL/EL로 화면 데이터 바인딩
- Filter/Listener, MVC 패턴 기초 구조

4) 동적 사이트 개발 (Spring 실전)
- Spring 핵심 모듈과 동작, Spring MVC
- 기본 커뮤니티 CRUD, 댓글
- 파일 업로드/다운로드
- Spring Security, OAuth2 로그인
- JWT Access/Refresh 인증 구조
- REST API 설계 및 리팩터링
- WebSocket/STOMP 기반 채팅 기능
- SSR/CSR 구조 이해
- 테스트코드

5) 배포 및 운영 기초
- Docker 기반 배포 흐름
- 실제 배포 환경 이슈 대응 관점
- 서비스 구조 문서화 및 유지보수 관점 정리




[✨ 수업의 강점]
✅ 개념 + 실습 동시 진행 (강의자료 제공)
설명만 하고 끝내지 않습니다.
핵심 개념 설명 후, 같은 시간에 바로 함께 구현합니다.

✅ 암기보다 구현 중심 학습
문법을 외우는 데서 그치지 않고, 스스로 기능을 만들어내는 실력을 목표로 합니다.

✅ 핵심 먼저, 확장은 단계적으로
매 수업 핵심 개념을 먼저 잡고,
필요한 확장 내용은 프로젝트 흐름에 맞춰 연결합니다.

✅ 전체 개발 흐름 통합 학습
프론트 + 백엔드 + DB + 인증 + 배포까지 실제 서비스 관점으로 연결해 학습합니다.

✅ 결과가 남는 커리큘럼
매 회차 결과물을 만드는 방식으로 진행합니다.
추상적인 이해가 아니라, 눈에 보이는 성장을 확인할 수 있습니다.

✅ 6개월 성장 목표 기반 지도
무경험자도 꾸준히 따라오면,
혼자서 기능을 설계하고 구현할 수 있는 수준을 목표로 합니다.

✅ 코딩을 목적이 아닌 수단으로
취업 준비에만 머무르지 않고,
직접 서비스와 프로젝트를 만들 수 있는 실행력을 키웁니다.

✅ 유연한 수강 운영 +  질문 지원
선입금 없이 회차별로 진행 가능하며, 중단/재개도 협의할 수 있습니다.
수업 외 실습 중 막히는 부분은  시간외에도  질문하면 가능한 빠르게 피드백합니다.
(보통 2시간 이내)




[🗓️ 수강 운영 예시]
- 주 1회 또는 주 2회 (협의)
- 회차별 목표 기능을 명확히 정해서 진행
- 진도형 + 프로젝트형 병행 가능
- 필요한 파트만 선택 수강 가능 (예: JWT 집중, Spring Security 집중)
