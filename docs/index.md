## 🛠️ 기술 스택

- **Language**: Java 17+, JavaScript (ES6+)
- **Backend**: Spring Boot 3.x, Spring Security, Spring Data JPA
- **Frontend**: Thymeleaf, HTML/CSS, jQuery, Vanilla JS
- **Auth**: Session, JWT (Access/Refresh), OAuth2
- **Realtime**: WebSocket + STOMP
- **DB**: MySQL / H2
- **Build**: Gradle
- **Deploy**: Docker
- **VCS**: Git, GitHub, GitKraken
