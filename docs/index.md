설명 문서는 `docs/` 아래에, 실습 코드는 루트 프로젝트 폴더에 유지합니다.

## 사용 방법

- 사이트에서는 `docs/<프로젝트>/` 경로로 설명 문서를 봅니다.
- 실습 코드는 루트의 원본 폴더(`java_basic/`, `javascript_basic/` 등)에서 확인합니다.
