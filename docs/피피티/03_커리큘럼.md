## 수업에 대한 약속

1. 비전공자도 따라올 수 있게 기초부터 단계적으로 진행합니다.
2. 설명만 하지 않고, 수업 중 직접 구현을 끝냅니다.
3. 템플릿 복붙이 아니라 코드 구조를 이해하고 작성하게 합니다.
4. 매 단원마다 리뷰를 통해 작동 코드에서 좋은 코드로 개선합니다.
5. 질문은 막히는 지점 중심으로 바로 해결합니다.
6. 최종적으로 문서화 가능한 프로젝트 결과물을 남깁니다.
