## 선택 심화 과정

<div style="height: 2px; background: #7f1d1d; margin: 10px 0 16px;"></div>

- Spring Security 내부 동작 심화
- JWT Access/Refresh 토큰 설계 심화
- OAuth2 로그인 흐름 및 커스터마이징
- REST API 리팩터링과 테스트 코드 강화
- STOMP 채팅 기능 확장 및 운영 고려사항
- Docker 배포 파이프라인 고도화

---

## 이런 분께 추천

1. 비전공자지만 개발을 제대로 시작하고 싶은 분
2. 설명형 수업이 아니라 구현형 수업이 필요한 분
3. 취업용 포트폴리오를 완성도 있게 만들고 싶은 분
4. 사이드/외주 프로젝트를 직접 진행하고 싶은 분
