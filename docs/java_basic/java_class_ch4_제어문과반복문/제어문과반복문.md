# 4장. 제어문과 반복문

## 학습 목표
- 조건문과 반복문의 용도를 구분한다.
- `if`, `switch`, `for`, `while`의 기본 구조를 익힌다.
- `break`, `continue`의 흐름 제어를 이해한다.

---

## 1. 조건문

### if / else if / else
- 조건식 결과에 따라 실행 경로를 나눈다.
- 복잡한 조건은 메서드로 분리해 가독성을 높인다.

### switch
- 하나의 값을 여러 경우로 분기할 때 유용하다.
- `break` 누락 시 의도치 않은 fall-through가 발생할 수 있다.

---

## 2. 반복문

### for
- 반복 횟수가 명확할 때 사용한다.

### while
- 반복 횟수가 불명확하고 조건 중심일 때 사용한다.

### do-while
- 최소 1회 실행이 보장된다.

---

## 3. 흐름 제어
- `break`: 반복/분기를 즉시 종료
- `continue`: 현재 회차를 건너뛰고 다음 반복 진행

---

## 4. 실무 관점 팁
- 중첩이 깊어지면 조건을 메서드로 분리한다.
- 무한 루프 방지를 위해 종료 조건을 명확히 둔다.
- 루프 내부에서 불필요한 연산을 줄여 성능을 개선한다.

---

## 정리
- 제어문은 "조건 판단", 반복문은 "반복 처리"에 최적화된 도구다.
- 핵심은 문법보다 흐름 설계와 가독성이다.
