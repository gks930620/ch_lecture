# 참고. 최신 문법과 스레드 학습 가이드

## 문서 목적
- 최신 Java 문법과 멀티스레드 주제를 한 번에 조망하기 위한 안내 문서다.
- 세부 이론은 각 단일 문서에서 학습하고, 이 문서는 전체 흐름을 잡는 데 사용한다.

---

## 학습 순서 권장
1. 최신 문법 기초 (`var`, `record`, `switch` 표현식, 패턴 매칭)
2. 함수형 문법(람다/스트림)과 최신 문법의 연결 이해
3. 멀티스레드 기본 개념(스레드 생명주기, 동기화, Executor)
4. 동시성 문제 해결 전략(공유 상태 최소화, 불변 객체, 스레드 안전 자료구조)

---

## 핵심 관점
- 최신 문법: 코드 가독성과 의도 전달을 개선하는 도구
- 스레드: 성능 최적화 도구이자 동시에 안정성 리스크가 큰 영역
- 결론적으로 "짧은 코드"보다 "예측 가능한 코드"가 우선이다.

---

## 실무 적용 체크리스트
- 팀의 JDK 표준 버전과 빌드/배포 환경이 일치하는가?
- 새 문법 도입이 코드 이해도를 실제로 높이는가?
- 스레드 코드에서 공유 자원 접근 규칙이 명확한가?
- 종료 처리, 예외 처리, 타임아웃 정책이 포함되어 있는가?

---

## 정리
- 최신 문법과 스레드 학습은 별개가 아니라 연결된 주제다.
- 문법은 생산성을, 동시성 설계는 안정성을 결정한다.
