# 16장. Java 최신 문법 개요

## 학습 목표
- 최근 Java에서 자주 쓰이는 문법 개선점을 파악한다.
- 코드 가독성과 안정성을 높이는 기능을 이해한다.
- 실무 도입 시 고려사항을 정리한다.

---

## 1. 대표 개선 포인트
- `var`: 지역 변수 타입 추론
- `switch` 표현식: 더 안전한 분기 작성
- 텍스트 블록: 멀티라인 문자열 가독성 향상
- `record`: 데이터 중심 불변 객체 간결화

---

## 2. 패턴 매칭 계열
- `instanceof` 패턴 매칭으로 형변환 코드 단순화
- `switch` 패턴 매칭으로 분기 로직 가독성 향상

---

## 3. 컬렉션/문자열 편의 기능
- `List.of`, `Set.of`, `Map.of`로 불변 컬렉션 생성
- `String` 유틸 메소드 확장으로 문자열 처리 간결화

---

## 4. 도입 시 체크리스트
- 프로젝트 JDK 버전과 배포 런타임 버전 일치 여부 확인
- 팀 코딩 컨벤션(예: `var` 사용 범위) 합의
- 새 문법의 가독성 이점이 명확할 때 적용

---

## 정리
- 최신 문법은 코드량 감소보다 "명확성 향상"을 목표로 도입하는 것이 효과적이다.
