# 3장. 연산자

## 학습 목표
- Java의 주요 연산자 종류를 구분한다.
- 연산자 우선순위와 결합 방향을 이해한다.
- 실수하기 쉬운 포인트를 정리한다.

---

## 1. 연산자 분류
- 산술 연산자: `+`, `-`, `*`, `/`, `%`
- 증감 연산자: `++`, `--`
- 비교 연산자: `==`, `!=`, `>`, `<`, `>=`, `<=`
- 논리 연산자: `&&`, `||`, `!`
- 대입 연산자: `=`, `+=`, `-=`, `*=`, `/=`
- 조건(삼항) 연산자: `조건 ? 값1 : 값2`
- 비트/시프트 연산자: `&`, `|`, `^`, `~`, `<<`, `>>`, `>>>`

---

## 2. 핵심 개념
- 정수 나눗셈은 소수점이 버려진다.
- `&&`, `||`는 단락 평가(short-circuit)를 수행한다.
- `==`는 기본형 값 비교, 참조형은 주소 비교가 기본이다.
- 문자열 내용 비교는 `equals()`를 사용한다.

---

## 3. 우선순위와 괄호
- 우선순위를 외우기보다, 애매하면 괄호로 의도를 명확히 한다.
- 복합식을 단순화하면 디버깅이 쉬워진다.

---

## 정리
- 연산자는 문법보다 "의도 전달"이 중요하다.
- 가독성 높은 식을 작성하는 습관이 오류를 크게 줄인다.
