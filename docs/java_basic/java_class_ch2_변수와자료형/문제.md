# 문제

`ch2` 범위(기본 자료형, 참조 자료형, 형변환, 상수/리터럴/var) 종합 문제입니다.

---

## A. 기초 확인

1. `byte`, `short`, `int`, `long`의 크기와 범위를 표로 정리하시오.
2. `float`와 `double`의 정밀도 차이를 설명하고, 기본 실수 리터럴 타입이 무엇인지 쓰시오.
3. 다음 코드에서 컴파일 오류가 나는 이유를 설명하시오.

```java
byte a = 10;
byte b = 20;
byte c = a + b;
```

4. `String`이 primitive가 아닌 이유를 설명하시오.
5. 배열이 참조 자료형이라는 것을 보여주는 예제를 작성하시오.

---

## B. 코드 작성

1. 정수 2개를 입력받아 합/차/곱/몫/나머지를 출력하는 프로그램을 작성하시오.
2. 반지름(`double`)을 입력받아 원의 넓이와 둘레를 출력하시오.
3. `String` 배열 5개를 만들고, null이 아닌 값만 출력하시오.
4. `int[]` 배열의 합계, 평균, 최대값, 최소값을 구하시오.
5. `String` 2개를 입력받아 내용이 같은지 비교하시오(`==` 사용 금지).

---

## C. 형변환 집중

1. `long -> int`, `double -> int`, `int -> byte` 변환 예제를 작성하고 값 손실 결과를 출력하시오.
2. `char ch = 'A';`에서 다음을 출력하시오.
- 문자 코드값
- 다음 문자
- 소문자 변환 결과
3. 정수 나눗셈과 실수 나눗셈의 결과 차이를 같은 입력으로 비교 출력하시오.
4. `Integer` 언박싱 시 `NullPointerException`이 발생하는 예제를 만들고 방어 코드를 작성하시오.

---

## D. String/배열 심화

1. 10,000번 문자열을 이어붙이는 코드에서 `String +`와 `StringBuilder` 성능 차이를 비교하시오.
2. 2차원 배열(행마다 길이가 다른 jagged array)을 만들고 전체 합을 구하시오.
3. 배열 복사(`clone`, `Arrays.copyOf`, `System.arraycopy`)를 각각 구현하고 결과를 비교하시오.
4. 참조 복사와 값 복사의 차이를 `int[]`와 사용자 정의 객체 배열로 설명하시오.

---

## E. 상수/리터럴/var

1. `static final` 상수 5개를 정의하고(예: 세율, 최대 재시도 횟수), 네이밍 규칙을 지켜 선언하시오.
2. 정수 리터럴을 2진/8진/10진/16진으로 같은 값을 표현하시오.
3. `var`를 썼을 때 가독성이 좋아지는 사례 2개, 나빠지는 사례 2개를 코드로 작성하시오.
4. `final` 참조 변수와 객체 불변성의 차이를 설명하는 예제를 작성하시오.

---

## F. 챌린지

1. 사용자에게 문자열 숫자를 입력받아 `int`로 변환한 뒤 합계를 구하시오.
- 잘못된 입력은 예외 처리로 건너뛰고 로그를 남기시오.
2. 문자열 배열에서 중복을 제거하고 길이 기준 내림차순 정렬하여 출력하시오.
3. 학생 점수 배열(`int[]`)을 입력받아 평균, 표준편차를 계산하시오.
- 중간 계산 타입 선택 이유(`int`, `double`)를 주석으로 설명하시오.

---

## 제출 체크리스트

1. 경계값 테스트를 했는가 (`MIN_VALUE`, `MAX_VALUE`, 0, null)?
2. 문자열 비교에 `equals`를 사용했는가?
3. 축소 형변환에서 손실 가능성을 의식하고 처리했는가?
4. `var` 사용이 코드 가독성을 해치지 않는가?
