# 2.1 기본 자료형 (Primitive Types)

## 학습 목표
- Java의 기본 자료형 8가지를 구분한다.
- 각 타입의 범위와 용도를 이해한다.
- 리터럴 표기와 주의사항을 익힌다.

---

## 1. 기본 자료형 종류

정수형:
- `byte`, `short`, `int`, `long`

실수형:
- `float`, `double`

문자형:
- `char`

논리형:
- `boolean`

---

## 2. 선택 기준
- 일반 정수 계산: `int`
- 큰 정수(금액, 카운트 범위 큼): `long`
- 일반 실수 계산: `double`
- 참/거짓 상태: `boolean`
- 한 글자 표현: `char`

---

## 3. 리터럴 규칙
- `long` 리터럴은 보통 `L` 접미사 사용
- `float` 리터럴은 `F` 접미사 필요
- 문자형은 작은따옴표, 문자열은 큰따옴표 사용

---

## 4. 주의사항
- 실수는 근사값이라 비교 시 오차가 생길 수 있다.
- 정수 연산은 범위를 넘으면 오버플로우가 발생할 수 있다.

---

## 정리
- 기본 자료형은 성능과 메모리 관점에서 핵심이다.
- 타입의 범위와 리터럴 규칙을 정확히 알고 써야 안전하다.
