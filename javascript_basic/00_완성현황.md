## 💡 사용 방법

### 강사용
1. 각 챕터의 `강의노트_X장_XXX.md` 파일을 먼저 읽고 내용 숙지
2. `partXX_XXX.html` 파일을 브라우저에서 열어 예제 확인
3. 강의 시 파일 주석을 참고하여 라이브 코딩
4. 학생들과 함께 버튼을 클릭하며 결과 확인

### 학생용
1. 각 HTML 파일을 순서대로 열어보기
2. 버튼을 클릭하여 동작 확인
3. 개발자 도구(F12)를 열어 코드 확인
4. 코드를 수정해보며 실험
