🚀 28.1-28.2: JavaScript란? & 변수 선언

1. JavaScript란?

JavaScript (JS)는 웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어입니다.

웹 기술의 3요소

  • HTML: 구조 (뼈대) 🏗️
  • CSS: 디자인 (외관) 🎨
  • JavaScript: 동작 (기능) ⚡

JavaScript로 할 수 있는 것

  • ✅ 버튼 클릭, 입력값 검증 등 이벤트 처리
  • ✅ HTML 내용 동적 변경
  • ✅ 애니메이션 효과
  • ✅ 서버와 통신 (AJAX)
  • ✅ 게임, 계산기, 타이머 등 구현

2. JavaScript 작성 위치

① 인라인 JavaScript

<!-- HTML 태그 내 -->
<button onclick="alert('클릭!')">버튼</button>

② 내부 JavaScript

<script>
  function sayHello() {
    alert('Hello!');
  }
</script>

③ 외부 JavaScript (권장) ⭐

<!-- HTML에서 연결 -->
<script src="script.js"></script>

3. 변수 (Variables)

변수는 데이터를 저장하는 상자와 같습니다.

변수 선언 키워드 (3가지)

키워드 재할당 재선언 스코프 권장도
var ✅ 가능 ✅ 가능 함수 스코프 ❌ (구식)
let ✅ 가능 ❌ 불가 블록 스코프 ✅ 추천
const ❌ 불가 ❌ 불가 블록 스코프 ✅ 추천

4. let - 재할당 가능한 변수

// let 변수 선언
let name = '홍길동';
let age = 25;
let isStudent = true;

// 값 변경 가능
name = '김철수';
age = 30;

console.log(name); // '김철수'
console.log(age); // 30

사용 시기: 값이 변경될 수 있는 변수

5. const - 재할당 불가능한 상수

// const 변수 선언
const PI = 3.14159;
const MAX_SIZE = 100;
const API_URL = 'https://api.example.com';

// 값 변경 시도 (❌ 에러 발생!)
PI = 3.14; // TypeError: Assignment to constant variable

사용 시기: 값이 변경되지 않는 상수

💡 팁: 기본적으로 const를 사용하고, 꼭 필요한 경우에만 let 사용!

6. var - 구식 변수 선언 (사용 지양)

// var 변수 선언 (ES6 이전 방식)
var oldName = '홍길동';
var oldName = '김철수'; // 재선언 가능 (문제 발생 가능!)

// let/const와 달리 함수 스코프
if (true) {
  var x = 10;
}
console.log(x); // 10 (블록 밖에서도 접근 가능)

⚠️ var는 사용하지 마세요! let과 const를 사용하세요.

7. 변수 명명 규칙

✅ 올바른 변수명

let userName = '홍길동'; // 카멜 케이스 (권장)
let user_name = '홍길동'; // 스네이크 케이스
let age = 25;
let _private = '값';
let $jquery = '값';
const MAX_VALUE = 100; // 상수는 대문자 (관례)

❌ 잘못된 변수명

let 2name = '값'; // 숫자로 시작 불가
let user-name = '값'; // 하이픈(-) 사용 불가
let user name = '값'; // 공백 사용 불가
let class = '값'; // 예약어 사용 불가

📌 명명 규칙 정리

  • ✅ 영문자, 숫자, _, $ 만 사용 가능
  • ✅ 첫 글자는 영문자, _, $ 만 가능 (숫자 불가)
  • ✅ 대소문자 구분 (name ≠ Name)
  • ✅ 의미있는 이름 사용 (x보다는 age, userName 등)
  • ✅ 카멜 케이스 사용 (userName, isLoggedIn)
  • ❌ 예약어 사용 불가 (let, const, if, for 등)

8. 실전 예제

// 사용자 정보
const USER_ROLE = 'admin';
let userName = '홍길동';
let userAge = 25;
let isLoggedIn = false;

// 값 변경
isLoggedIn = true;
userAge = userAge + 1; // 26

// 출력
console.log(`이름: ${userName}`);
console.log(`나이: ${userAge}`);
console.log(`로그인: ${isLoggedIn}`);

✅ 실습 과제

  1. let과 const를 사용하여 자신의 정보를 변수로 저장하기 (이름, 나이, 직업 등)
  2. 변수 값을 변경하고 console.log()로 출력하기
  3. alert()와 prompt()를 사용하여 사용자 입력 받기
  4. 계산기 기능 만들기 (두 숫자를 입력받아 더하기)

💡 JavaScript 작성 팁

  • ✅ 기본적으로 const 사용, 필요시에만 let
  • ✅ var는 절대 사용하지 마세요!
  • ✅ 의미있고 명확한 변수명 사용
  • ✅ 카멜 케이스 사용 (userName, isActive)
  • ✅ 상수는 대문자와 언더스코어 (MAX_SIZE, API_URL)
  • ✅ console.log()로 디버깅하는 습관 들이기
