🎨 27.1-27.2: CSS란? & CSS 적용 방법

1. CSS란?

CSS (Cascading Style Sheets)는 HTML 요소의 스타일을 정의하는 언어입니다.

  • Cascading: 여러 스타일 규칙이 계단식으로 적용됨
  • Style: 색상, 크기, 레이아웃 등을 지정
  • Sheets: 스타일 규칙의 모음

CSS의 역할

  • ✅ HTML: 구조 (뼈대)
  • ✅ CSS: 디자인 (외관)
  • ✅ JavaScript: 동작 (기능)

CSS 기본 문법

선택자 {
  속성: 값;
  속성: 값;
}

예시)
h1 {
  color: red;
  font-size: 24px;
}

2. CSS 적용 방법 (3가지)

① 인라인 스타일 (Inline Style)

HTML 태그의 style 속성에 직접 작성

<h1 style="color: red; font-size: 30px;">제목</h1>
<p style="background: yellow; padding: 10px;">내용</p>

이 문단은 인라인 스타일이 적용되었습니다!

장점: 즉시 적용, 우선순위 가장 높음

단점: 유지보수 어려움, 재사용 불가

⚠️ 권장하지 않음 - 테스트 목적으로만 사용

② 내부 스타일시트 (Internal CSS)

HTML 문서의 <head> 안에 <style> 태그로 작성

<head>
  <style>
    h1 { color: blue; }
    p { font-size: 16px; }
  </style>
</head>

이 문단은 내부 스타일시트가 적용되었습니다!

장점: 한 페이지 내 모든 요소에 적용 가능

단점: 다른 페이지에서 재사용 불가

사용 시기: 단일 페이지에만 적용할 스타일이 있을 때

③ 외부 스타일시트 (External CSS) ⭐ 추천

별도의 .css 파일을 만들고 HTML에서 연결

/* style.css 파일 */
h1 { color: green; }
p { line-height: 1.6; }

/* HTML에서 연결 */
<head>
  <link rel="stylesheet" href="style.css">
</head>

이 문단은 외부 스타일시트가 적용되었습니다!

장점: 여러 페이지에서 재사용, 유지보수 용이, 코드 분리

단점: 추가 HTTP 요청 발생 (미미함)

✅ 가장 권장하는 방법!

3. CSS 적용 방법 비교

방법 작성 위치 우선순위 재사용성 권장도
인라인 HTML 태그 내 ⭐⭐⭐ (가장 높음) ❌ (불가) ❌
내부 <head> 내 <style> ⭐⭐ (중간) △ (같은 페이지만) △
외부 별도 .css 파일 ⭐ (낮음) ✅ (여러 페이지) ✅ (추천)

4. CSS 우선순위 (Specificity)

여러 CSS 규칙이 충돌할 때 적용 우선순위

우선순위 (높음 → 낮음)
  1. 인라인 스타일 (style 속성)
  2. ID 선택자 (#id)
  3. 클래스/속성/가상 선택자 (.class, [attr], :hover)
  4. 태그 선택자 (h1, p, div)
  5. 전체 선택자 (*)

!important 키워드를 사용하면 모든 우선순위를 무시하고 최우선 적용!

/* 우선순위 예시 */
p { color: black; } /* 1순위 */
.text { color: blue; } /* 2순위 - 클래스가 태그보다 우선 */
#title { color: green; } /* 3순위 - ID가 클래스보다 우선 */
<p style="color: red;"> /* 4순위 - 인라인이 가장 우선 */
p { color: yellow !important; } /* 5순위 - !important가 최우선 */
