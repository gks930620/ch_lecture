🎯 27.3: CSS 선택자 (Selectors)

선택자란?

CSS를 적용할 HTML 요소를 선택하는 방법입니다.

선택자를 잘 활용하면 원하는 요소에 정확하게 스타일을 적용할 수 있습니다!

1️⃣ 태그 선택자 (Type Selector)

HTML 태그 이름으로 선택합니다.

p { color: blue; }
h1 { font-size: 32px; }
div { background: yellow; }

이 문단은 태그 선택자로 스타일이 적용됩니다.

모든 p 태그에 동일한 스타일이 적용됩니다.

특징: 해당 태그를 가진 모든 요소에 적용됩니다.

2️⃣ 클래스 선택자 (Class Selector)

.클래스명 형태로 선택합니다.

.highlight { background: yellow; }
.blue-text { color: blue; }
.large-text { font-size: 20px; }

이 문단은 .highlight 클래스가 적용되었습니다.

이 문단은 .blue-text 클래스가 적용되었습니다.

여러 클래스를 동시에 적용할 수 있습니다!

특징: 재사용 가능, 여러 요소에 적용, 하나의 요소에 여러 클래스 사용 가능

3️⃣ ID 선택자 (ID Selector)

#아이디명 형태로 선택합니다.

#special { background: purple; color: white; }
#header { padding: 20px; }
이 div는 #special ID가 적용되었습니다.
ID는 페이지당 하나만 사용해야 합니다!

특징: 고유함 (페이지당 하나만), 클래스보다 우선순위 높음

4️⃣ 전체 선택자 (Universal Selector)

* 기호로 모든 요소를 선택합니다.

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

용도: CSS 초기화(reset) 시 주로 사용

5️⃣ 그룹 선택자 (Group Selector)

여러 선택자에 동일한 스타일을 적용할 때 ,로 구분합니다.

h1, h2, h3 {
  color: navy;
  font-weight: bold;
}

.btn1, .btn2, .btn3 {
  padding: 10px;
  border-radius: 5px;
}

6️⃣ 자손 선택자 (Descendant Selector)

A B 형태로, A 요소 하위의 모든 B 요소를 선택합니다.

.container .item {
  border: 2px solid blue;
}
Container
Item 1 (직접 자식)
Item 2 (손자)

특징: 깊이에 상관없이 하위의 모든 요소가 선택됩니다.

7️⃣ 자식 선택자 (Child Selector)

A > B 형태로, A 요소의 직접적인 자식 B 요소만 선택합니다.

.parent > .child {
  background: lightgreen;
}
Parent
Child (직접 자식 - 스타일 적용 ✅)
Grandchild (손자 - 스타일 미적용 ❌)

특징: 바로 아래 자식만 선택 (손자는 선택 안 됨)

8️⃣ 인접 형제 선택자 (Adjacent Sibling)

A + B 형태로, A 요소 바로 다음의 B 요소를 선택합니다.

h3 + p {
  color: red;
  font-weight: bold;
}

제목

이 문단은 h3 바로 다음이므로 스타일 적용 ✅

이 문단은 두 번째이므로 스타일 미적용 ❌

9️⃣ 일반 형제 선택자 (General Sibling)

A ~ B 형태로, A 요소 다음의 모든 형제 B 요소를 선택합니다.

h3 ~ p {
  color: green;
}

🔟 속성 선택자 (Attribute Selector)

HTML 속성을 기반으로 선택합니다.

[type="text"] { border: 2px solid green; }
[type="email"] { border: 2px solid blue; }
[href^="https"] { color: green; } /* https로 시작 */
[href$=".pdf"] { color: red; } /* .pdf로 끝남 */
[class*="btn"] { padding: 10px; } /* btn 포함 */

📊 선택자 우선순위

선택자 점수 예시
인라인 스타일 1000점 style="color: red;"
ID 선택자 100점 #header
클래스/속성/가상 선택자 10점 .box, [type="text"], :hover
태그 선택자 1점 div, p, h1
전체 선택자 0점 *
💡 점수 계산 예시:
  • #header .nav li = 100 + 10 + 1 = 111점
  • .container .box .item = 10 + 10 + 10 = 30점
  • div p = 1 + 1 = 2점

✅ 실습 과제

  1. 태그, 클래스, ID 선택자를 모두 사용하여 페이지 만들기
  2. 자손 선택자와 자식 선택자의 차이 테스트하기
  3. 속성 선택자를 사용하여 다양한 input 스타일링하기
  4. 선택자 우선순위를 직접 테스트해보기

💡 선택자 사용 팁

  • ✅ 클래스 선택자를 주로 사용하세요 (재사용성 높음)
  • ✅ ID 선택자는 JavaScript용으로 남겨두세요
  • ✅ 선택자는 가능한 간단하게 작성하세요
  • ✅ 의미있는 클래스 이름을 사용하세요
  • ❌ 지나치게 복잡한 선택자는 피하세요
  • ❌ 전체 선택자(*)의 과도한 사용은 지양하세요
