32.1-32.4: jQuery 소개와 기본 사용법

1. jQuery란?

jQuery는 JavaScript를 더 쉽고 간결하게 사용할 수 있도록 만든 라이브러리입니다.

jQuery의 특징

  • ✅ "Write Less, Do More" - 적게 쓰고 더 많이 하자!
  • ✅ 간결한 문법으로 DOM 조작
  • ✅ 크로스 브라우저 호환성 (구형 브라우저 지원)
  • ✅ 강력한 선택자 (CSS 선택자 그대로 사용)
  • ✅ 애니메이션 효과 쉽게 구현
  • ✅ AJAX 쉽게 구현

jQuery의 역사

  • 2006년 존 레식(John Resig)이 개발
  • 한때 웹 개발의 표준 라이브러리
  • 현재는 Vanilla JS가 많이 발전했지만, 여전히 많은 프로젝트에서 사용 중

2. jQuery 설치 및 사용법

① CDN 방식 (권장) ⭐

HTML 파일에 다음 코드를 추가합니다 (보통 </body> 태그 직전):

<!-- jQuery 3.7.1 CDN -->
<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

② 다운로드 방식

jquery.com에서 파일을 다운로드하여 프로젝트에 포함:

<script src="js/jquery-3.7.1.min.js"></script>
💡 이 페이지는 이미 jQuery를 로드했습니다!
콘솔에 typeof jQuery를 입력하면 "function"이 출력됩니다.

3. $ 기호 - jQuery의 핵심

$는 jQuery의 별칭(alias)입니다. jQuery와 $는 동일합니다!

// 두 가지 방법 모두 동일
jQuery('div').hide();
$('div').hide(); // 주로 이렇게 사용

$는 함수이며, 다음과 같은 용도로 사용됩니다:

  • 요소 선택
  • HTML 생성
  • 문서 준비 이벤트

4. 문서 준비 이벤트 $(document).ready()

DOM이 완전히 로드된 후에 코드를 실행하도록 보장합니다.

전체 버전

$(document).ready(function() {
  // DOM이 준비된 후 실행될 코드
  console.log('문서 준비 완료!');
});

단축 버전 (많이 사용) ⭐

$(function() {
  // DOM이 준비된 후 실행될 코드
  console.log('문서 준비 완료!');
});

ES6 화살표 함수 버전

$(() => {
  console.log('문서 준비 완료!');
});
💡 $(document).ready() vs window.onload
  • $(document).ready(): DOM 구조만 로드되면 실행 (빠름) ⚡
  • window.onload: 이미지 등 모든 리소스 로드 후 실행 (느림) 🐢

5. jQuery 기본 선택자

CSS 선택자와 동일한 방식으로 요소를 선택할 수 있습니다!

선택자 예시 설명
태그 $('p') 모든 <p> 태그 선택
클래스 $('.box') class="box"인 모든 요소
ID $('#header') id="header"인 요소
속성 $('[type="text"]') type이 text인 요소
자식 $('ul > li') ul의 직접 자식 li
자손 $('div p') div 안의 모든 p

6. Vanilla JS vs jQuery 비교

예제 1: 요소 선택

Vanilla JavaScript jQuery 
document.getElementById('box')
document.querySelector('.box')
document.querySelectorAll('.box')
 
$('#box')
$('.box')
$('.box')

예제 2: 텍스트 변경

Vanilla JavaScript jQuery 
document.getElementById('box')
  .textContent = '새 텍스트';
 
$('#box').text('새 텍스트');

예제 3: 클릭 이벤트

Vanilla JavaScript jQuery 
document.querySelector('.btn')
  .addEventListener('click',
    function() {
      alert('클릭!');
    });
 
$('.btn').click(function() {
  alert('클릭!');
});

결론: jQuery가 훨씬 간결합니다! 🎉

7. 실전 예제

텍스트 변경 테스트

원본 텍스트입니다.

요소 숨기기/보이기

이 박스를 숨기거나 보일 수 있습니다!

클래스 추가/제거

이 텍스트의 스타일이 변경됩니다.

코드

// 문서 준비
$(() => {
  // 텍스트 변경
  $('#btn-change-text').click(() => {
    $('#demo-text').text('jQuery로 변경된 텍스트!');
  });

  // 요소 숨기기
  $('#btn-hide').click(() => {
    $('#demo-box').hide();
  });

  // 요소 보이기
  $('#btn-show').click(() => {
    $('#demo-box').show();
  });

  // 클래스 추가
  $('#btn-add-class').click(() => {
    $('#demo-class').css({
      'background': 'yellow',
      'font-weight': 'bold',
      'color': 'red'
    });
  });
});
