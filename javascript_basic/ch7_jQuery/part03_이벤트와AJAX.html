⚡ 32.12-32.15: jQuery 이벤트 & AJAX

1. 이벤트 핸들링

주요 이벤트 메소드

메소드 설명 예제
click() 클릭 이벤트 $('.btn').click(fn)
dblclick() 더블 클릭 $('.item').dblclick(fn)
hover() 마우스 호버 $('.box').hover(fn1, fn2)
focus() 포커스 받을 때 $('input').focus(fn)
blur() 포커스 잃을 때 $('input').blur(fn)
keyup() 키를 뗄 때 $('input').keyup(fn)
change() 값이 변경될 때 $('select').change(fn)
submit() 폼 제출 시 $('form').submit(fn)

이벤트 테스트

나를 클릭하세요!

나를 더블클릭하세요!

나에게 마우스를 올려보세요!

// 클릭 이벤트
$('.btn').click(function() {
  alert('버튼 클릭!');
});

// 더블 클릭
$('.item').dblclick(function() {
  $(this).css('background', 'yellow');
});

// 호버 (마우스 올릴 때, 벗어날 때)
$('.box').hover(
  function() { $(this).addClass('highlight'); },
  function() { $(this).removeClass('highlight'); }
);

2. on() 메소드 - 통합 이벤트 핸들러 ⭐

on()은 모든 이벤트를 처리할 수 있는 강력한 메소드입니다!

// 기본 사용법
$('.btn').on('click', function() {
  console.log('클릭됨');
});

// 여러 이벤트 동시 처리
$('input').on('focus blur', function(e) {
  console.log(e.type + ' 이벤트 발생');
});

// 이벤트 위임 (동적 요소에도 적용)
$('#list').on('click', 'li', function() {
  $(this).toggleClass('active');
});

off() - 이벤트 제거

// 특정 이벤트 제거
$('.btn').off('click');

// 모든 이벤트 제거
$('.btn').off();

3. 폼 이벤트

폼 이벤트 실습

// focus 이벤트
$('input').focus(function() {
  $(this).css('border-color', 'blue');
});

// change 이벤트
$('select').change(function() {
  let value = $(this).val();
  console.log('선택: ' + value);
});

// submit 이벤트 (폼 제출 방지)
$('form').submit(function(e) {
  e.preventDefault(); // 기본 동작 방지
  alert('폼 제출 완료!');
});

4. jQuery AJAX 기초

AJAX (Asynchronous JavaScript And XML)는 페이지 새로고침 없이 서버와 통신하는 기술입니다.

$.ajax() - 가장 강력한 방법

$.ajax({
  url: 'https://api.example.com/data',
  method: 'GET', // 또는 'POST', 'PUT', 'DELETE'
  data: { id: 123 },
  dataType: 'json',
  success: function(response) {
    console.log('성공:', response);
  },
  error: function(xhr, status, error) {
    console.log('에러:', error);
  }
});

$.get() - GET 요청 간편 버전

$.get('https://api.example.com/data', function(data) {
  console.log('데이터:', data);
});

$.post() - POST 요청 간편 버전

$.post('https://api.example.com/data', {
  name: '홍길동',
  age: 25
}, function(response) {
  console.log('응답:', response);
});

AJAX 실습 - 공개 API 호출

JSONPlaceholder API를 사용하여 실제 데이터를 가져옵니다.

실제 예제 코드

// GET 요청
$.ajax({
  url: 'https://jsonplaceholder.typicode.com/posts/1',
  method: 'GET',
  success: function(data) {
    console.log('제목:', data.title);
    console.log('내용:', data.body);
  }
});

// POST 요청
$.post('https://jsonplaceholder.typicode.com/posts', {
  title: '새 게시글',
  body: '게시글 내용입니다.',
  userId: 1
}).done(function(response) {
  console.log('생성된 ID:', response.id);
});

  3. 폼 검증 기능 구현하기 (focus, blur, submit 이벤트 활용)
  4. AJAX를 사용하여 API 데이터 가져오기
  5. 할 일 목록 앱 만들기 (이벤트 + DOM 조작 통합)

💡 주요 팁

  • ✅ 이벤트 위임을 활용하면 동적 요소에도 이벤트 적용 가능
  • ✅ e.preventDefault()로 기본 동작 방지
  • ✅ $(this)로 이벤트가 발생한 요소에 접근
  • ✅ AJAX는 Promise 방식(.done, .fail, .always)도 지원
  • ✅ 최신 프로젝트에서는 fetch API나 axios 사용 고려
