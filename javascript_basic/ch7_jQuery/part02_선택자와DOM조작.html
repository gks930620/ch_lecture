1. jQuery 선택자 완벽 정리

기본 선택자

선택자 예제 설명
태그 $('div') 모든 div 선택
클래스 $('.box') class="box" 선택
ID $('#header') id="header" 선택
전체 $('*') 모든 요소 선택
다중 $('div, .box, #header') 여러 선택자를 동시에

필터 선택자

선택자 예제 설명
:first $('li:first') 첫 번째 li
:last $('li:last') 마지막 li
:even $('tr:even') 짝수 번째 (0,2,4...)
:odd $('tr:odd') 홀수 번째 (1,3,5...)
:eq(n) $('li:eq(2)') 3번째 li (0부터 시작)
:gt(n) $('li:gt(2)') 3번째보다 큰 인덱스
:lt(n) $('li:lt(2)') 3번째보다 작은 인덱스
:not() $('li:not(.active)') 제외하기

2. 계층 선택자

선택자 예제 설명
자손 $('div p') div 안의 모든 p
자식 $('div > p') div의 직접 자식 p만
형제 $('.box').siblings() 형제 요소들
다음 형제 $('.box').next() 바로 다음 형제
이전 형제 $('.box').prev() 바로 이전 형제
부모 $('.box').parent() 부모 요소
조상 $('.box').parents() 모든 조상 요소

3. DOM 조작 - 텍스트/HTML

text() - 텍스트 내용

// 텍스트 가져오기
let txt = $('#demo').text();

// 텍스트 설정하기
$('#demo').text('새로운 텍스트');

html() - HTML 내용

// HTML 가져오기
let html = $('#demo').html();

// HTML 설정하기
$('#demo').html('<strong>굵은 텍스트</strong>');

val() - 입력 값

// input 값 가져오기
let value = $('#name').val();

// input 값 설정하기
$('#name').val('홍길동');

4. DOM 조작 - 요소 추가/제거

메소드 설명 예제
append() 마지막 자식으로 추가 $('div').append('<p>끝</p>')
prepend() 첫 번째 자식으로 추가 $('div').prepend('<p>처음</p>')
after() 요소 뒤에 추가 $('p').after('<span>뒤</span>')
before() 요소 앞에 추가 $('p').before('<span>앞</span>')
remove() 요소 제거 $('.box').remove()
empty() 자식 요소만 제거 $('div').empty()

요소 추가 테스트

기존 내용


// 요소 추가 예제
$('#list').append('<li>새 항목</li>'); // 마지막 추가
$('#list').prepend('<li>새 항목</li>'); // 처음 추가
$('.box').after('<div>뒤에</div>'); // 뒤에 추가
$('.box').before('<div>앞에</div>'); // 앞에 추가
$('.box').remove(); // 요소 제거

5. 속성 조작

attr() - 속성 가져오기/설정

// 속성 가져오기
let src = $('img').attr('src');

// 속성 설정
$('img').attr('src', 'new-image.jpg');
$('a').attr('href', 'https://www.google.com');

// 여러 속성 동시 설정
$('img').attr({
  'src': 'image.jpg',
  'alt': '이미지 설명',
  'width': '300'
});

removeAttr() - 속성 제거

$('img').removeAttr('width');
