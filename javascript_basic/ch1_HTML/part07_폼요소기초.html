📝 26.8: 폼 요소 기초

1. <form> 태그

사용자 입력을 받아서 서버로 전송하는 컨테이너

<form action="/submit" method="POST">
  폼 요소들...
</form>
주요 속성:
  • action: 데이터를 전송할 URL
  • method: 전송 방식 (GET, POST)
  • name: 폼의 이름

2. <input> 태그

다양한 타입의 입력을 받을 수 있는 태그

3. <button> 태그

클릭 가능한 버튼

button vs input type="button"
  • <button>: 내부에 HTML 요소 포함 가능 (이미지, 아이콘 등)
  • <input type="button">: 텍스트만 가능

4. <textarea> 태그

여러 줄의 텍스트 입력

<textarea rows="5" cols="30">기본 텍스트</textarea>

5. <select> 태그 (드롭다운)

여러 옵션 중 하나를 선택

Ctrl/Cmd를 누르고 클릭하면 여러 개 선택 가능

6. 종합 예제: 회원가입 폼

회원가입

7. 주요 input 속성

속성 설명
placeholder 입력 힌트 표시
required 필수 입력 필드
disabled 입력 비활성화
readonly 읽기 전용
maxlength 최대 글자 수
minlength 최소 글자 수
pattern 정규식 패턴 검증
value 기본값 설정

✅ 실습 과제

  1. 설문조사 폼을 만들어보세요 (이름, 연령대, 만족도 등)
  2. 상품 주문 폼을 만들어보세요 (상품 선택, 수량, 배송지 등)
  3. 문의하기 폼을 만들어보세요 (이름, 이메일, 제목, 내용)
