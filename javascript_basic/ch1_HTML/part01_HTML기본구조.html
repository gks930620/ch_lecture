📚 26.1-26.2: HTML이란? & HTML 문서 구조

🔍 HTML이란?

HTML (HyperText Markup Language)은 웹 페이지의 구조를 만드는 마크업 언어입니다.

  • HyperText: 링크를 통해 다른 문서로 이동할 수 있는 텍스트
  • Markup: 태그를 사용하여 문서의 구조를 표시
  • Language: 웹 브라우저가 이해할 수 있는 언어

📝 HTML 문서의 기본 구조

DOCTYPE: HTML5 문서임을 선언

html: HTML 문서의 루트 요소

head: 메타데이터(문서 정보)를 포함

body: 실제로 화면에 표시되는 콘텐츠

🎯 주요 메타 태그

  • <meta charset="UTF-8"> - 문자 인코딩 설정 (한글 깨짐 방지)
  • <meta name="viewport"> - 모바일 반응형 설정
  • <title> - 브라우저 탭에 표시되는 제목

✅ 실습 확인

이 페이지는 올바른 HTML5 구조로 작성되었습니다:

  1. ✅ DOCTYPE 선언
  2. ✅ html 태그 (lang="ko" 속성)
  3. ✅ head 섹션 (메타 정보, 스타일)
  4. ✅ body 섹션 (실제 콘텐츠)

브라우저 개발자 도구(F12)를 열어서 HTML 구조를 확인해보세요!
