📦 26.11-26.12: div, span & 속성

1. <div> - 블록 레벨 컨테이너

Division(구역)의 약자로, 여러 요소를 그룹화하는 컨테이너입니다.

<div>
  <h2>제목</h2>
  <p>내용</p>
</div>

특징:

  • ✅ 블록 레벨 요소 (한 줄 전체를 차지)
  • ✅ 줄바꿈이 발생함
  • ✅ width, height, margin, padding 모두 적용 가능
  • ✅ 레이아웃을 만들 때 주로 사용
첫 번째 div (블록 레벨)
두 번째 div (블록 레벨)
세 번째 div (블록 레벨)

2. <span> - 인라인 레벨 컨테이너

텍스트의 일부분을 스타일링하거나 그룹화할 때 사용합니다.

<p>이것은 <span style="color: red;">빨간색</span> 텍스트입니다.</p>

특징:

  • ✅ 인라인 레벨 요소 (콘텐츠 크기만큼만 차지)
  • ✅ 줄바꿈이 발생하지 않음
  • ✅ width, height는 적용 불가 (display 속성 변경 시 가능)
  • ✅ 텍스트 일부에 스타일을 적용할 때 사용

이것은 빨간색 텍스트이고, 이것은 노란 배경이며, 이것은 파란색 큰 글씨입니다.

3. div vs span 비교

구분 div span
레벨 블록 (Block) 인라인 (Inline)
줄바꿈 O (자동 줄바꿈) X (줄바꿈 없음)
width/height 설정 가능 설정 불가
용도 레이아웃, 큰 구역 텍스트 일부 스타일링

4. 주요 속성 (Attributes)

📌 id 속성

요소에 고유한 식별자를 부여합니다. 페이지 내에서 유일해야 합니다!

<div id="header">헤더 영역</div>
<div id="main-content">메인 콘텐츠</div>
  • JavaScript에서 getElementById()로 선택
  • CSS에서 #id 선택자로 스타일 적용
  • 페이지 내 앵커 링크로 사용 (#id)

📌 class 속성

여러 요소에 공통 스타일을 적용할 때 사용합니다. 중복 가능!

<div class="box">박스 1</div>
<div class="box highlight">박스 2</div>
<div class="box">박스 3</div>
  • 하나의 요소에 여러 class 적용 가능 (공백으로 구분)
  • JavaScript에서 getElementsByClassName()으로 선택
  • CSS에서 .class 선택자로 스타일 적용
일반 박스
일반 박스
강조 박스 (여러 클래스)

📌 name 속성

주로 폼 요소에서 사용하며, 서버로 전송될 때 데이터의 키(key)가 됩니다.

<input type="text" name="username">
<input type="email" name="email">

📌 data-* 속성 (Custom Data Attributes)

사용자 정의 데이터를 저장할 때 사용합니다. JavaScript에서 활용!

<div data-user-id="12345" data-role="admin">사용자 정보</div>
<button data-product-id="100" data-price="50000">상품 구매</button>

📌 title 속성

요소에 마우스를 올렸을 때 툴팁을 표시합니다.

<span title="이것은 툴팁입니다">마우스를 올려보세요</span>
여기에 마우스를 올려보세요! 👆

📌 style 속성

요소에 인라인 스타일을 직접 적용합니다.

<div style="color: red; font-size: 20px; background: yellow;">스타일 적용</div>

⚠️ 주의: 인라인 스타일은 유지보수가 어려우므로, 가급적 CSS 파일로 관리하세요!

5. 종합 예제

사용자 프로필

이름: 홍길동
나이: 25세
직업: 개발자

<div id="user-card" class="card">
  <h3 class="card-title">사용자 프로필</h3>
  <p>
    이름: <span class="username">홍길동</span><br>
    나이: <span class="age" data-value="25">25세</span><br>
    직업: <span class="job" title="소프트웨어 개발자">개발자</span>
  </p>
  <div class="actions">
    <button class="btn btn-primary" data-action="profile">프로필 보기</button>
    <button class="btn btn-secondary" data-action="message">메시지 보내기</button>
  </div>
</div>

✅ 실습 과제

  1. div와 span을 사용하여 명함 디자인 만들기
  2. id, class, data-* 속성을 모두 활용한 상품 카드 만들기
  3. 여러 div로 그리드 레이아웃 구성하기
  4. span으로 텍스트 일부에 다양한 스타일 적용하기

💡 모범 사례

  • ✅ 의미있는 id와 class 이름 사용 (예: header, main-content, user-card)
  • ✅ 케밥 케이스(kebab-case) 또는 카멜 케이스(camelCase) 일관되게 사용
  • ✅ 시맨틱 태그를 우선 사용하고, 필요시에만 div/span 사용
  • ❌ 지나친 div 중첩 지양 (div hell)
  • ❌ 인라인 스타일 과다 사용 지양
