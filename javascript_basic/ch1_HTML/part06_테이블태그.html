📊 26.7: 테이블 태그

1. 기본 테이블 구조

주요 태그:

  • <table>: 테이블 전체를 감싸는 태그
  • <tr>: 테이블 행 (Table Row)
  • <td>: 테이블 데이터 셀 (Table Data)
  • <th>: 테이블 헤더 셀 (Table Header)
<table>
  <tr>
    <th>이름</th>
    <th>나이</th>
  </tr>
  <tr>
    <td>홍길동</td>
    <td>25</td>
  </tr>
</table>

예제: 간단한 테이블

이름 나이 직업
홍길동 25 개발자
김철수 30 디자이너
이영희 28 기획자

2. thead, tbody, tfoot

테이블을 논리적으로 구분하는 태그 (선택사항이지만 권장)

상품명 가격 수량
노트북 1,200,000원 2
마우스 30,000원 5
키보드 80,000원 3
합계 10개

3. colspan - 열 병합

여러 열을 하나로 합칠 때 사용

학생 정보
이름 학년 반
홍길동 3 1
colspan="3"으로 3개 열을 병합

4. rowspan - 행 병합

여러 행을 하나로 합칠 때 사용

과목 교수님 시간
컴퓨터공학 김교수 월 1-2교시
이교수 수 3-4교시
수학 박교수 화 1-2교시

✅ 실습 과제

  1. 본인의 주간 시간표를 만들어보세요 (rowspan, colspan 활용)
  2. 가계부 테이블을 만들어보세요 (수입/지출, 합계 포함)
  3. 상품 목록 테이블을 만들고 스타일링 해보세요

💡 테이블 사용 시 주의사항
  • 레이아웃 목적으로 테이블 사용 금지 (CSS Flexbox/Grid 사용)
  • 표 형태의 데이터를 표현할 때만 사용
  • 반응형 웹에서는 테이블이 잘 안 보일 수 있으므로 모바일 대응 필요
