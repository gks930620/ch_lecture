📝 26.3: 텍스트 태그

<h1> ~ <h6> 제목 태그

h1 - 가장 큰 제목

h2 - 두 번째 제목

h3 - 세 번째 제목

h4 - 네 번째 제목

h5 - 다섯 번째 제목
h6 - 가장 작은 제목

💡 팁: h1은 페이지당 하나만 사용하는 것이 SEO에 좋습니다!

<p> 문단 태그

이것은 문단(paragraph)입니다. 하나의 문단을 나타낼 때 사용합니다.

p 태그는 자동으로 위아래 여백이 생깁니다.

문단을 구분하는 가장 기본적인 태그입니다.

<span> 인라인 텍스트 태그

이것은 빨간색 텍스트이고, 이것은 노란 배경입니다.

span vs p: span은 인라인 요소, p는 블록 요소

<br> 줄바꿈 태그

첫 번째 줄
두 번째 줄
세 번째 줄

주의: br은 닫는 태그가 없습니다! (self-closing tag)

<hr> 수평선 태그

위 내용

아래 내용

용도: 주제 전환이나 섹션 구분에 사용

기타 텍스트 포맷팅 태그

<strong>: 굵은 텍스트 (중요함을 의미)

<b>: 굵은 텍스트 (단순 스타일)

<em>: 강조 텍스트 (기울임, 의미 있음)

<i>: 이탤릭 텍스트 (단순 스타일)

<mark>: 형광펜 효과

<small>: 작은 텍스트

<del>: 삭제된 텍스트

<ins>: 추가된 텍스트

<sub>: H2O (아래 첨자)

<sup>: X2 (위 첨자)

✅ 실습 과제

  1. 자기소개 페이지를 만들어보세요 (h1~h3, p 태그 사용)
  2. 중요한 부분은 strong, 강조할 부분은 em 사용
  3. hr로 섹션을 구분하세요
