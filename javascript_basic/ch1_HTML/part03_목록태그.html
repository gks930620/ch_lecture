📋 26.4: 목록 태그 (ul, ol, li)

1️⃣ <ul> - 순서 없는 목록 (Unordered List)

용도: 순서가 중요하지 않은 항목들을 나열할 때

예제: 좋아하는 음식

  • 피자
  • 치킨
  • 파스타
  • 햄버거

중첩된 목록

  • 과일
    • 사과
    • 바나나
    • 오렌지
  • 채소
    • 상추
    • 토마토
    • 당근

2️⃣ <ol> - 순서 있는 목록 (Ordered List)

용도: 순서가 중요한 항목들을 나열할 때

예제: 라면 끓이는 방법

  1. 물 550ml를 끓인다
  2. 면과 스프를 넣는다
  3. 4분 30초간 끓인다
  4. 맛있게 먹는다

type 속성으로 번호 스타일 변경

type="1" (기본값) - 숫자

  1. 첫 번째
  2. 두 번째
  3. 세 번째

type="A" - 대문자

  1. 첫 번째
  2. 두 번째
  3. 세 번째

type="a" - 소문자

  1. 첫 번째
  2. 두 번째
  3. 세 번째

type="I" - 로마 숫자 (대문자)

  1. 첫 번째
  2. 두 번째
  3. 세 번째

start 속성 - 시작 번호 지정

  1. 다섯 번째부터 시작
  2. 여섯 번째
  3. 일곱 번째

3️⃣ <dl>, <dt>, <dd> - 정의 목록

용도: 용어와 설명을 함께 표현할 때

HTML
웹 페이지의 구조를 만드는 마크업 언어
CSS
웹 페이지의 스타일을 꾸미는 스타일시트 언어
JavaScript
웹 페이지에 동적인 기능을 추가하는 프로그래밍 언어

✅ 실습 과제

  1. 나의 하루 일과를 순서 있는 목록(ol)로 작성하세요
  2. 좋아하는 영화 장르를 순서 없는 목록(ul)로 작성하고, 각 장르별로 영화 제목을 중첩 목록으로 추가하세요
  3. 프로그래밍 용어 3개를 정의 목록(dl)으로 작성하세요
