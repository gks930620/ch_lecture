🖼️ 26.6: 이미지 태그 (<img>)

1. 기본 이미지 삽입

필수 속성:

  • src: 이미지 파일의 경로
  • alt: 이미지 설명 (이미지가 안 보일 때 표시, 접근성 중요)
<img src="이미지경로.jpg" alt="이미지 설명">

샘플 이미지 (400x200)

2. 이미지 크기 조절

width와 height 속성

<img src="image.jpg" alt="설명" width="300" height="200">

width="300" height="200"

💡 팁: width나 height 중 하나만 지정하면 비율이 유지됩니다!
추천: CSS로 크기를 조절하는 것이 더 유연합니다.

3. 이미지 경로

📁 상대 경로

<img src="image.jpg" alt="같은 폴더">
<img src="images/photo.jpg" alt="하위 폴더">
<img src="../image.jpg" alt="상위 폴더">

🌐 절대 경로 (URL)

<img src="https://example.com/image.jpg" alt="인터넷 이미지">

인터넷 URL로 불러온 이미지

4. 이미지 스타일링

테두리 추가

둥근 이미지

그림자 효과

5. 클릭 가능한 이미지 (링크)

a 태그로 img를 감싸면 이미지가 링크가 됩니다.

<a href="https://www.example.com">
  <img src="image.jpg" alt="클릭 이미지">
</a>

이미지를 클릭하면 Google로 이동합니다

6. figure와 figcaption

이미지에 캡션(설명)을 추가할 때 사용합니다.

<figure>
  <img src="image.jpg" alt="설명">
  <figcaption>이미지 캡션</figcaption>
</figure>

7. 반응형 이미지

화면 크기에 따라 자동으로 조절되는 이미지

<img src="image.jpg" alt="반응형" style="max-width: 100%; height: auto;">

브라우저 창 크기를 조절해보세요!

8. 이미지 정렬

좌측 정렬

이미지가 왼쪽에 정렬되고 텍스트가 오른쪽으로 흐릅니다. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

우측 정렬

이미지가 오른쪽에 정렬되고 텍스트가 왼쪽으로 흐릅니다. Lorem ipsum dolor sit amet, consectetur adipiscing elit. Sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

중앙 정렬

✅ 실습 과제

  1. 자신의 프로필 사진을 원형으로 만들어보세요 (border-radius: 50%)
  2. 갤러리 페이지를 만들어보세요 (4~6개 이미지)
  3. 이미지에 hover 효과를 추가해보세요 (CSS transition 사용)
  4. figure와 figcaption을 사용하여 사진첩을 만들어보세요
🎯 이미지 파일 형식
  • JPG/JPEG: 사진, 풍경 (파일 크기 작음)
  • PNG: 투명 배경, 로고, 아이콘
  • GIF: 애니메이션
  • SVG: 벡터 이미지 (확대해도 깨지지 않음)
  • WebP: 차세대 이미지 형식 (용량 작고 화질 좋음)
