🎨 26.9: Input 타입 완벽 가이드

HTML5에서 제공하는 다양한 Input 타입을 알아봅시다

1. type="text" - 텍스트

가장 기본적인 텍스트 입력 필드

<input type="text" placeholder="홍길동" maxlength="20">

2. type="password" - 비밀번호

입력 내용이 ●●●로 표시됩니다

<input type="password" minlength="8">

3. type="email" - 이메일

이메일 형식 자동 검증 (@ 포함 확인)

<input type="email" required>

4. type="number" - 숫자

숫자만 입력 가능 (min, max, step 설정 가능)

<input type="number" min="18" max="100" step="1">

5. type="tel" - 전화번호

전화번호 입력 (모바일에서 숫자 키패드 자동 표시)

<input type="tel" pattern="[0-9]{3}-[0-9]{4}-[0-9]{4}">

6. type="url" - URL

URL 형식 자동 검증 (http:// 또는 https:// 필요)

<input type="url" placeholder="https://example.com">

7. type="search" - 검색

검색창 (X 버튼 자동 생성)

<input type="search" placeholder="검색어를 입력하세요">

8. type="date" - 날짜

날짜 선택기 (캘린더 자동 표시)

<input type="date" min="1900-01-01" max="2024-12-31">

9. type="time" - 시간

시간 선택 (시:분)

<input type="time" value="14:00">

10. type="datetime-local" - 날짜 + 시간

날짜와 시간을 동시에 선택

<input type="datetime-local">

11. type="month" - 월

년월 선택

<input type="month">

12. type="week" - 주

연도와 주차 선택

<input type="week">

13. type="color" - 색상

색상 선택기 (Color Picker)

색상을 선택하면 배경색이 변경됩니다!

<input type="color" value="#e91e63">

14. type="range" - 범위 슬라이더

슬라이더로 값 선택

<input type="range" min="0" max="100" value="50">

15. type="file" - 파일 선택

파일 업로드

<input type="file" accept="image/*">
<input type="file" multiple accept=".pdf,.doc">
accept 속성:
  • image/* : 모든 이미지 파일
  • video/* : 모든 비디오 파일
  • audio/* : 모든 오디오 파일
  • .pdf : PDF 파일만

16. type="checkbox" - 체크박스

여러 개 선택 가능

<input type="checkbox" name="hobby" value="reading"> 독서

17. type="radio" - 라디오 버튼

하나만 선택 가능 (같은 name 속성 필요)

<input type="radio" name="gender" value="male"> 남성

18. type="hidden" - 숨겨진 필드

화면에는 보이지 않지만 폼 전송 시 함께 전송됩니다

위에 숨겨진 input이 있습니다 (개발자 도구로 확인 가능)

<input type="hidden" name="user_id" value="12345">

19. type="submit" / type="reset" / type="button"

버튼 타입

<input type="submit" value="제출하기">
<input type="reset" value="초기화">
<input type="button" value="일반 버튼">

✅ 실습 과제

  1. 모든 input 타입을 사용한 종합 설문조사 폼 만들기
  2. 예약 시스템 폼 만들기 (날짜, 시간, 인원 등)
  3. 상품 등록 폼 만들기 (이름, 가격, 색상, 이미지 등)
  4. 체크박스와 라디오 버튼을 활용한 선호도 조사 폼 만들기
💡 꿀팁:
  • 모바일에서는 type에 따라 적절한 키보드가 자동으로 나타납니다
  • required 속성으로 필수 입력 필드를 지정할 수 있습니다
  • pattern 속성으로 정규식을 사용한 검증이 가능합니다
  • autofocus 속성으로 페이지 로드 시 자동 포커스를 줄 수 있습니다
