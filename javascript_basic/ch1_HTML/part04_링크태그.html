🔗 26.5: 링크 태그 (<a>)

1. 기본 링크

href 속성: 이동할 URL을 지정

구글로 이동
네이버로 이동

2. target 속성

_self (기본값): 현재 창에서 열기

현재 창에서 열기

_blank: 새 창(탭)에서 열기

새 창에서 열기

💡 보안 팁: _blank 사용 시 보안을 위해 rel="noopener noreferrer" 추가 권장

안전한 새 창 열기

3. 상대 경로 vs 절대 경로

절대 경로: 전체 URL

https://www.example.com/page.html

상대 경로: 현재 위치 기준

Part 1로 이동 (같은 폴더)
상위 폴더의 파일로 이동

4. 페이지 내 이동 (앵커)

id를 이용한 책갈피 기능

섹션 1로 이동 | 섹션 2로 이동 | 섹션 3로 이동

5. 이메일 링크

이메일 보내기

6. 전화번호 링크 (모바일)

7. 버튼 스타일 링크

구글 방문 GitHub 방문 맨 위로

8. 다운로드 링크

download 속성: 클릭 시 다운로드

HTML 파일 다운로드
